## 2. Prologue: Will it Cook?

### 2.1. Kitchen

- USE fridge **2 times** (`Antimatter`, `Liquid Helium Tank`, `Spare Moustache`, `Atomic Clock`, `Tea Leaves`, `Unidentified Blue Object`)
- >**ACHIEVEMENT:** Boop
  >- USE cat
- USE `Antimatter` on `Liquid Helium Tank` (`Liquid Helium with a sprinkle of Antimatter`)
- USE `Liquid Helium with a sprinkle of Antimatter` on pot
  - CHOOSE as you like
- USE your chosen inventory item on pot (`To-Do-List`)
- USE `To-Do-List`
  - CLICK on ? (in the bottom right corner)
- USE `Tea Leaves` OR `Unidentified Blue Object` on pot (depends on your choice during the dialogue)
- >**ACHIEVEMENT:** Rightful place
  >- USE `Spare Moustache` on cat
  >- USE cat (`Spare Moustache`)
- USE `Spare Moustache` on pot
- >**ACHIEVEMENT:** It did not cook
  >- USE `Atomic Clock` on pot

## 7. Episode 5: Triple Trouble

### 7.1. Bedroom

- EXIT east

### 7.2. Laboratory

- USE cake
- PICK UP pictures (`Birthday Photo`, `Selfie`)
- USE door
- PICK UP book (`Reality-Defying Book`)
  - TURN PAGE to last page
  - USE `Selfie` on illustration
    - **NOTE:** You switch rooms with this action.
  - EXIT (at the bottom of the screen)

### 7.3. Kitchen

- PICK UP drawing (from the fridge) (`Drawing`)
- TALK TO robotic eel
- TALK TO uncle Lee
- TALK TO witch (Leeanne)
- USE `Reality-Defying Book`
  - USE `Drawing` on author picture
- TALK TO robotic eel (`Sticker`)
  - **NOTE:** We need `Science, magic and wisdom`
- USE `Sticker` on `Birthday Photo`
- USE `Reality-Defying Book`
  - USE `Drawing` on illustration

### 7.4. Bathroom

- USE books (`Collection of Rules`)
- PICK UP shaving creme (`Shaving Cream`)
- USE `Reality-Defying Book`
  - USE `Birthday Photo` on author picture
    - **NOTE:** You need to have put the sticker on the photo, so you become yourself from the future
  - USE `Collection of Rules` on rule #3
    - SELECT Clarke's Law #3 "Any sufficiently advanced technology..."
  - USE `Collection of Rules` on rule #4
    - SELECT Uncle Lee's Law #3 "Look inside yourself"
- TALK TO Leeanne
  - "I'll need you to perform the WRATH..."
- TALK TO mirror
  - "If you're from the future..."
  - CHOOSE any words
- TALK TO uncle Lee
  - "Uncle Lee, I need you to participate..."
    - **NOTE:** You need to have `Clarke #3` put in the `Reality-Defying Book` to get uncle Lee to help

### 7.5. Kitchen

- USE `Shaving Creme` on raviol-o-matic(TM) (`Raviolo`)
- USE `Reality-Defying Book`
  - `Collection of Rules` on rule #2
    - SELECT Uncle Lee's Law #2 "Substitute. Always."
- USE `Raviolo` on Leeanne
- >**ACHIEVEMENT:** They will pay
- >**ACHIEVEMENT:** Champion
