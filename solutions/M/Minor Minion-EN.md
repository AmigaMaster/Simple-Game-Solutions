# Minor Minion - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 03.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/minor-minion)

## 1. Gargoyle's Hallway

- WALK to the right (you talk to a mouse)
  - **NOTE:** The symbols on the letter
- PICK UP wooden block (`Wood`)
- USE ladder to EXIT down

## 2. Slab Hallway

- PICK UP glove (`Glove`)
- PICK UP sock (`Sock`)
- PICK UP plane (`Plane`)
- EXIT down

## 3. Torture Chamber

- PICK UP shrunken head (`Shrunken Head`)
- USE `Wood` on iron maiden (`Spike`)
- PICK UP torch (`Torch`)
- EXIT up **times**

## 4. Gargoyle's Hallway

- USE `Spike` on right Gargoyle (`Gargoyle Head`)
- USE `Gargoyle Head` on mouse (`Cheese`)
- USE `Torch` on Gaxtur (`Earwax`)

## 5. Slab Hallway

- USE `Spike` on slab
- EXIT down using the new ladder

### 5.1. Altar

- PICK UP ink (`Ink`, `Feather`)
- PICK UP potion (`Potion`)
- USE drawer
  - You saw these symbols on the letter in the signature
  - USE the same order:
    1. Right
    2. Bottom right
    3. Top
    4. Bottom left
    5. Left
- PICK UP **all** stones (`Rune`, `Rune`, `Rune`, `Rune`)
- PICK UP book (`Book`)
- EXIT up and down the right ladder

## 6. Torture Chamber

- USE `Sock` on `Cheese` (`Key`, `Flail of bad smell`)
- USE `Key` on chest
- USE `Book` on princess
- USE **all four** `Rune` on princess
- USE `Earwax` on `Ink` (`Colored Earwax`)
- USE `Colored Earwax` on `Wood` (`Crayon`)
- USE `Crayon` on princess (`Netiv Rune`)
- USE `Plane` on `Wood` (`Wood shavings`)
- USE `Wood shavings` on `Shrunken Head` (`Head`)

## 7. Gargoyle's Hallway

- USE `Glove` on Gaxtur
- USE `Glove` on `Head` (`Doll`)

## 8. Torture Chamber

- USE `Doll` on princess (`Grot Rune`)
- USE `Netiv Rune` on `Grot Rune` (the runes with the yellow symbols) (Spell: Netiv Grot)

## 9. Gargoyle's Hallway

- WALK to the left
- USE Netiv Grot (an icon in the top menu bar)
- USE left hole

### 9.1. Tunnel

- WALK left until you automatically flee

## 10. Gargoyle's Hallway

- EXIT down and come back here
- USE `Flail of bad smell` on Gaxtur (`Bloody Flail`)
- USE `Bloody Flail` on hole (`Rat`)
- USE `Plane` on `Rat` (`Sand rat`)

## 11. Torture Chamber

- USE `Sand Rat` on princess (`Appras Rune`)
- USE `Netiv Rune` on `Appras Rune` (Spell: Netiv Appras)
- USE Netiv Appras

## 12. Gargoyle's Hallway

- EXIT up

## 13. Entrance

- WALK on the left side of the villager
- TALK TO villager
- EXIT up

## 14. Loft

- USE `Plane` on winch (`Chain`)

## 15. Entrance

- USE Netiv Appras
- PICK UP pitchfork (`Pitchfork`)

## 16. Gargoyle's Hallway

- USE `Pitchfork` on eyes (of the left gargoyle) (`Ruby`)
- USE `Ruby` on `Chain` (`Necklace`)

## 17. Torture Chamber

- USE `Necklace` on princess (`Canco Rune`)
- USE `Canco Rune` on `Appras Rune` (Spell: Canco Appras)

## 18. Entrance

- USE Canco Appras
- EXIT west
