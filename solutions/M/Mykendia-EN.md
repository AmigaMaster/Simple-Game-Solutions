# Mykendia Part 1-3 - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 30.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

## 1. Part 1

### 1.1. Castle of Mykendia

- PICK UP feather (from the bridge) (`Feather`)
- USE `Knife` on Blossom (`Perfume Blossom`)
- EXIT west

### 1.2. Village

- USE `Flute` on water pot
- PICK UP nest (`Coin`)
- ENTER Chez Jaques

#### 1.2.1. Chez Jaques

- TALK TO alcoholic
- USE `Coin` on alcoholic
- TALK TO waiter
- USE `Perfume Blossom` on glass
- PICK UP potatoes (`Potato`)
- EXIT east, west

### 1.3. Land of Trolls

- You automatically return to the map

### 1.4. Disguise Shop

- TALK TO Trillon (`List`)
- USE `List` on Falk (yourself)
- EXIT east

### 1.5. Matthew's Farm

- LOOK AT cow
- USE `Feather` on cow
- PICK UP dung (`Horsedung`)
- PICK UP pile of straw (`Straw`)
- EXIT east

### 1.6. Dark Forest

- TALK TO sawmiller
- USE rope (to the right of the house) **12 times**
- USE `List` on sawmiller (`Troll's Teeth`)
- PICK UP mushroom (behind the pile of logs) (`Mushroom`)
- EXIT east

### 1.7. Disguise Shop

- USE `Troll's Teeth` on Trillon
- USE `Straw` on Trillon
- USE `Horsedung` on Trillon
- USE `Knife` on `Potato` (`Troll's Nose`)
- USE `Troll's Nose` on Trillon (`Troll Dress`)

### 1.8. Land of Trolls

- You automatically walk through

### 1.9. Troll Market

- USE `Mushroom` on Yok the trader (`Shovel`)
- EXIT west

### 1.10. Cave

- USE `Shovel` on dirt

#### 1.10.1. Cave Entrance

- USE `Shovel` on patch of dirt
- USE string
- USE `List` on altar
- PICK UP The Laughter Stone

## 2. Part 2

### 2.1. Tent of Miss Fortuna

- PICK UP strange things inside the tent (`Bottle`)
- EXIT east

### 2.2. Water Tower

- USE `Bottle` on brook (`Bottle with Water`)
- ENTER tower

#### 2.2.1. Tower

- LOOK AT machine
- USE machine
  - SELECT the lake on the right of the painting
- EXIT west **2 times**

### 2.3. Sun Tower

- ENTER tower

#### 2.3.1. Tower

- PICK UP wooden horse (`Wooden Horse`)
- USE machine
  - SELECT the sun on the right
- EXIT west **2 times**

### 2.4. Fire Tower

- ENTER tower

#### 2.4.1. Tower

- USE `Bottle with Water` on machine (`Bottle`)
- USE machine
  - SELECT the fire
- EXIT west, east

### 2.5. Where the Tower Lights Meet

- USE shovel on spot (`Stone`)
- EXIT east

### 2.6. Water Tower

- USE `Bottle` on brook (`Bottle with Water`)

### 2.7. Village

- TALK TO boy
- USE `Wooden Horse` on boy (`Marbles`)
- PICK UP stone (from the path to the left house) (`Rope`)
- EXIT east

### 2.8. Fire Tower

- USE `Stone` on strange symbol
- ENTER cave

#### 2.8.1. Cave Entrance

- USE `Bottle with Water` on wagon (`Bottle`)
- USE `Rope` on machine
- USE `Stone` on machine
- USE wagon

#### 2.8.2. Rails

- USE `Flute` on huge stalacsomething

#### 2.8.3. Lake

- USE `Flute` on the colored stalactites in the following order
  1. Blank (first from the left side)
  2. yellow
  3. red
  4. black
  5. blue
  6. pink
  7. green
  8. white

#### 2.8.4. Waterfall

- USE `Knife` on small tree (`Trunk`)
- USE `Trunk` on hole (`Stone`)

### 2.9. Demon's Tower

- USE `Marbles` on demon guard

#### 2.9.1. Jail

- USE `Shovel` on thin wall (lower left back wall)
- USE `Bottle` on barrel (`Bottle with Acid`)
- USE `Bottle with Acid` on door

## 3. Part 3

### 3.1. House

- TALK TO Mackany
- PICK UP bag (from the left shelf) (`Seeds`)
- USE `Bottle` on tears (`Lizard Tears`)
- EXIT west

### 3.2. Witch House

- TALK TO witches **2 times**
- PICK UP chicken (under the pot) (`Chicken`)
- PICK UP invisibility powder (behind the witch on the right) (`Invisibility Powder`)
- EXIT west

### 3.3. Forest Pond

- USE `Invisibility Powder` on Falk (yourself)
- PICK UP fairy wings (`Fairy Wing`)
- EXIT east

### 3.4. Foot of the Volcano

- USE `Seeds` on volcano soil
- EXIT up

#### 3.4.1. Giant

- USE `Chicken` on dog house
- PICK UP Giant's hair (`Giant's Hair`)
- EXIT west **2 times**

### 3.5. Witch House

- USE `Giant's Hair` on witches
- USE `Lizard Tears` on witches
- USE `Fairy Wing` on witches (`Magic Broom`)

### 3.6. Foot of the Volcano

- USE `Magic Broom` on Falk
