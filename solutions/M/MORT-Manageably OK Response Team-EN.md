# MORT: Manageably OK Response Team - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 02.10.2024 | Amiga Master (AmigaMaster-NG@t-online.de) |

Play the game directly in your browser from the [Homepage](https://rootbound.itch.io/mort-manageably-ok-response-team) or download the game from [archive.org](https://archive.org/details/mort-manageably.-ok.-response.-team.-7z)

## 1. Rooftop

>Green

- OPEN **both** exhaust caps
- OPEN panel
- TAKE circuit connector (`Circuit connector`)

## 2. Basement

>Purple

- USE power wheel
- TAKE circuit connector (`Circuit connector`)
- USE `Circuit connector` with Helper Drone
- USE drone with (upper) connector slot
- USE breaker box
- USE breaker switches
- USE lift (to go up)

## 3. Robot Guard Floor

- USE panel **2 times** (`Metal Strip`)
- USE lift (to go down)

## 4. Basement

- USE `Metal Strip` with breaker door (`Shiny Door`)
- USE lift

## 5. Robot Guard Floor

- USE `Shiny Door` with guard bot
- USE lift (to go up)

## 6. Chute Bottom Floor

- USE panel

## 7. Rooftop

>Green

- USE lift (to go down)
- USE `Circuit connector` with connector slot
- USE lever (to move the magnet to the left)
- USE button
- TAKE magnet (`Freight Magnet`)
- USE (right) lift (to go down)

## 8. Chute Bottom Floor

>Purple

- USE drone with chute bottom

## 9. Chute Top Floor

> Green

- USE drone with switch
- USE (left) lift (to go down)

## 10. Blue Guard Bot Floor

- WALK to the blue guard bot to lure it to the left lift
- USE drone with switch **2 times**
- USE (left) lift to go up and make your way to the Chute Top Floor

## 11. Chute Top Floor

- USE `Freight Magnet` with drone
  - USE Drone with chute top

## 12. Chute Bottom Floor

>Purple

- USE drone with panel (to put the magnet in the panel)
- USE magnet
- TAKE magnet (`Freight Magnet`)
- USE (left) lift

## 13. Broken Lift Floor

- **QUICK:** Walk to the right side to get out of the lift
- USE `Freight Magnet` with magnet hookup
- USE lever
- USE (red) button
- USE (green) call button
- USE (right) lift (down)

## 14. Chute Bottom Floor

- USE (right) lift (up)

## 15. Broken Lift Floor

- **QUICK:** Leave the lift to the left side
- USE drone with (red) button (crate must be on the right side)
  - USE drone with magnet
  - RETAKE ITEM from drone (`Freight magnet`)

## 16. Chute Top Floor

> Green

- USE (left) lift (to go down)

## 17. Blue Guard Bot Floor

- USE (right) lift (down)

## 18. Motion Sensor Floor

- USE lever

## 19. Broken Lift Floor

>Purple

- USE (left) lift (up)

## 20. Heat Sensor Floor

- USE drone with `Freight magnet`
  - USE drone (still carrying the magnet) with chute bottom

## 21. Chute Top Floor

>Green

- USE lever (crane to the right side)
- USE drone (still carrying the magnet)
  - USE drone with magnet hookup
  - USE drone with lever
  - USE drone with button
  - USE drone with panel
- USE panel

## 22. Heat Sensor Floor

>Purple

- USE panel
- USE (right) lift (up)

## 23. Chute Top Floor

>Green

- USE (left) lift (down)

## 24. Cell Floor

- USE drone with panel (in the middle, above Pink Mort) (`Circuit Connector`)
  - USE drone with (right) chute entrance to get to the left side
  - USE drone `Circuit Connector` with (left) connector slot

>Purple

- Each time the yellow robot is still moving to the center:
  - **QUICK:** OPEN panel
  - **QUICK:** TAKE panel (`Freight Magnet`)
  - **QUICK:** USE `Tarnished door` with panel
- USE lever (move the magnet to the right side)
- USE button
- USE drone with (left) connector slot (`Circuit Connector`)
  - USE drone with (left) chute entrance
  - USE drone with (right) connector slot
  - USE drone with (right) connector slot
- USE `Freight Magnet` with drone
- USE drone with (left) chute entrance
  - USE drone with magnet hookup
  - USE drone with (blue) guard bot

>Green

- USE lever (move the magnet to the left)
- USE button
- USE `Grenade` with holding cell

>Purple

- USE `Grenade` with holding cell
