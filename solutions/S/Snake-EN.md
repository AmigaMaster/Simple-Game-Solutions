# Snake - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 14.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/mags-snake)

## 1. Lake

- TALK TO stone (on the right)
- PICK UP stick (in front of you) (`Stick`) (2/2)
- EXIT north

## 2. Forest Crossing

- DROP (right-click) `Stick` next to the tree
- EXIT east

## 3. Cupboard

- USE (black) pole
  - You want to see the wizard (3/5)
- TALK TO door

### 3.1. Inside Cupboard

- TALK TO blue cup (5/10)
- PICK UP (upper) red flower pot (`Flower Pot`) (2/12)
- EXIT south, southwest

## 4. Forest Crossing

- DROP `Flower Pot` near the tree (8/20)
- EXIT west

## 5. Gate

- TALK TO wolf
- USE pole (2/22)
- PICK UP button (`Button`)
- TALK TO pole (1/31)
- TALK TO wolf
  - "Why isn't this pole working?"
- EXIT east

## 6. Forest Crossing

- DROP `Button` next to the flowerpot (`Hand`) (20/51)
- PICK UP flowerpot (`Flowerpot`)
- PICK UP fork (`Fork`) (2/53)
- EXIT south

## 7. Lake

- TALK TO rock
  - "Have you been by the gate lately?" (`Circuit Board`) (2/55)
- PICK UP brown plant (`Tobacco Leaf`) (2/57)
- PICK UP hemp (the plant right of the path to he north) (`Hemp Leaf`) (2/59)
- USE `Flowerpot` on lake (`Flowerpot with Water`) (1/60)

## 8. Gate

- USE `Tobacco Leaf` on `Hemp Leaf` (`Tobacco with Hemp`) (6/66)
- USE `Tobacco with Hemp` on wolf (5/71)
- USE `Circuit Board` on pole (7/78)
- TALK TO pole
  - "Look..."

## 9. Cupboard

- TALK TO pole
  - "I'm here to see the big wizard guy."
- USE door

### 9.1. Inside Cupboard

- TALK TO Cup
  - "Can you protect me..." (5/83)
- EXIT south

## 10. Cupboard

- PICK UP coffee plant (`Coffee Beans`) (2/85)

## 11. Gate

- TALK TO pole
  - "Look..." (10/95)
- EXIT north

## 12. Table

- EXIT east

## 13. Lake

- USE lake (2/97)

## 14. Table

- PICK UP stocking (from the table) (`Stocking`) (2/99)
- TALK TO rabbit (`Cup`) (2/101)
- PICK UP glue
- TALK TO rabbit
  - Ask about the glue

## 15. Lake

- The stone talks to you (5/106)
  - **NOTE:** You need to have talked to the rabbit about the glue
- USE `Flowerpot with Water` on yourself
- TALK TO rock
  - "Can I borrow you..."

## 16. Gate

- EXIT west

## 17. Map

- WALK TO brown brown patch on the right

## 18. Giant

- TALK TO giant
- **QUICK:**
  - USE `Coffee Beans` on spot under the club
- TALK TO giant
- **QUICK:**
  - PICK UP beans (`Coffee Powder`) (3/109)
  - **NOTE:** You need to be very quick as the pick up menu takes some time to pop up.
- TALK TO rock
  - "Can I borrow you..."
- EXIT west

## 19. Lake

- DROP rock
- USE lake

## 20. Table

- USE `Coffee Powder` on rabbit (`Glue`) (5/114)

## 21. Cupboard

- PICK UP coffee plant (`Coffee Beans`)

## 22. Lake

- USE `Flowerpot with Water` on yourself
- TALK TO rock
  - "Can I borrow you..."

## 23. Map

- WALK TO crashed spaceship in the south

## 24. Spaceship

- TALK TO **both** aliens
- PICK UP coffee maker (`Coffee Maker`) (3/117)
- USE `Glue` on `Coffee Beans` (`Glued Beans`) (3/120)
- USE `Glued Beans` on tray next to the alien
- **QUICK:**
  - PICK UP lemon (`Lemon`) (4/124)
- TALK TO rock
  - "Can I borrow you..."
- EXIT northwest

## 25. Giant

- TALK TO giant
- **QUICK:**
  - USE `Coffee Beans` on spot under the club
- TALK TO giant
- **QUICK:**
  - PICK UP beans (`Coffee Powder`) (3/109)
  - **NOTE:** You need to be very quick as the pick up menu takes some time to pop up.
- TALK TO rock
  - "Can I borrow you..."

## 26. Cupboard

- TALK TO pole
  - "I'm here to see the big wizard guy."
- USE door

### 26.1. Inside Cupboard

- USE `Coffee Maker` on socket on the left
- USE `Stocking` on coffee maker
- USE `Cup` on coffee maker
- USE `Flowerpot with Water` on coffee maker
- USE `Coffee Powder` on coffee maker
- USE `Fork` on `Lemon` (`Bio Battery`) (6/132)
- USE `Bio Battery` on coffee maker (`Coffee`) (18/150)
- USE `Coffee` on blue cup
