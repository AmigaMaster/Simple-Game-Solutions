# SAW 2 - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 23.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/904-saw-2)

## 1. Cell

- USE rope (`Chunks of Rope`)
- PICK UP tape recorder (from Zep's chest) (`Tape Recorder`)
- PICK UP piece of paper (from the ground) (`Piece of Paper`)
- EXAMINE `Piece of Paper`
- PICK UP newspaper (`Newspaper`)
- EXAMINE `Newspaper`
- PICK UP cassette tape 1 (in the newspaper pile) (`Cassette 1`)
- USE `Cassette 1` with `Tape Recorder`
- TALK TO Zepp
  - Ask about cassette
  - Ask about newspaper
- EXAMINE box (above the chair)
  - **NOTE:** Number `3`
  - RETURN
- EXAMINE hole (in the wall)
- PICK UP object (in the hole) (`Tape 1`)
- EXAMINE Zepp
  - PICK UP red object (`Key Number 1`)
- EXIT east

## 2. Kitchen

- PICK UP shovel (top left corner) (`Shovel`)
- EXAMINE drawer (middle of the left cabinet)
- PICK UP drawer (`Knife`)
- PICK UP small box (the black part in the broken furniture) (`Matches`)
- PICK UP note (refrigerator) (`Note`)
- EXAMINE `Note`
- USE door of the kiln
- USE hollow (from the oven)

## 3. Cellar

- PICK UP saw (`Saw`)
- EXAMINE box (back wall)
  - **NOTE:** Number `1`
  - USE `Key Number 1` with lock
  - PICK UP triangle 1 (`Triangle 1`)
  - PICK UP key 1 (`Key 1)
  - RETURN
- USE `Tape 1` with video
- EXAMINE substance (on the wall)
- USE `Chunks of Rope` with substance
- USE `Matches` with rope
- USE marked brick (about at the height where the chain is attaches to the wall, in the middle between the TV and Allen)
- PICK UP tape (`Cassette 2`)
- PICK UP key (a very small hotspot) (`Ancient Key`)
- USE `Cassette 2` with `Tape Recorder`
- USE `Tape 1` with video
  - ENTER `1=0`
- PICK UP envelope (in Allen's hand) (`Envelope`)
- EXAMINE `Envelope` (`Cassette Allen`)
- USE `Cassette Allen` with `Tape Recorder`
- PICK UP hypodermic (on the ground) (`Hypodermic`)
- EXIT south (in the center of the screen)

## 4. Kitchen

- EXIT south
  - EXAMINE background (top of the door)
  - USE `Knife` with background
  - EXAMINE numeric sequence
    - **NOTE:** `Add the worthless` and on the TV it said `1=0`
    - You need to add all the numbers but leave the 1’s out
    - ENTER `56`
    - RETURN
- EXAMINE the box (next to the sink)
  - ENTER `056`
- EXIT south
  - USE little metallic door
  - USE `Ancient Key` with lock
  - USE right arrow

## 5. Control Room

- EXAMINE model of bathroom (on the right) (`Mobile Phone`)
- EXAMINE `Mobile Phone`
  - USE button between the green and red buttons
  - EXIT
- EXAMINE briefcase (on the left)
  - EXAMINE money
  - EXAMINE documents (`Documents (Briefcase)`)
  - PICK UP paper (from the top) (`Paper (Briefcase)`)
  - PICK UP tape (`Tape 2`)
  - EXAMINE `Documents (Briefcase)`
    - **NOTE:** The X on the drawing on page 4
    - EXIT
  - RETURN
- EXIT south

## 6. Kitchen

- EXAMINE wall tile (above the sink)
  - **NOTE:** `468`
- USE wall tile
- EXAMINE `Mobile Phone`
  - DIAL `468`
  - USE green button
- EXAMINE (left) box
  - PICK UP triangle 2 (`Triangle 2`)
  - PICK UP key 2 (`Key 2`)
  - RETURN
- USE hollow (stove)

## 7. Cellar

- WALK TO zone (bottom right corner)
- USE `Shovel` with zone
- PICK UP key 3 (`Key Number 3`)
- USE `Tape 2` with video
- EXAMINE the message.
- EXIT south, west

## 8. Cell

- EXAMINE box
  - USE `Key Number 3` with lock
  - PICK UP triangle 3 (`Triangle 3`)
  - PICK UP key 3 (`Key 3`)

## 9. Control Room

- EXAMINE box
  - USE `Triangle 1` with mechanism
  - USE `Triangle 2` with mechanism
  - USE `Triangle 3` with mechanism
  - USE `Triangle 4` with mechanism
  - USE mechanism
  - PICK UP hypodermic (`Hypodermic (A)`)
  - PICK UP key (`Key 4`)
  - RETURN

## 10. Cell

- USE `Key 3` with left hand chain
- USE `Key 4` with right hand chain
- USE `Key 1` with right foot shackle (on his foot, not on the chain!)
- USE `Key 2` with left foot shackle

## 11. Office

- USE `Saw` with chain (on the door, a ridiculous hard to click hotspot beside other hotspots)
- USE door
- USE hood
