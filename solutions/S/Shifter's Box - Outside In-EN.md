# Shifter's Box - Outside In - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 25.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1137-shifter-s-box-outside-in)

## 1. Path

- USE box
  - "Try to open the box."
- USE tree (`Branch`)
- USE `Branch` on box

## 2. Platform

- PICK UP bucket (from the tall pipe on the right) (`Bucket`)
- LOOK AT `Bucket` (`Pipe Cap`)
- PICK UP cap (on the ground on the left) (`Pipe Cap`)
- USE `Bucket` on drain
- USE `Pipe Cap` on pipe (above the drain)
- USE `Pipe Cap` on pipe (on the far right)
- USE pipe valve
- PICK UP bucket (`Bucket`)
- USE `Bucket` on contraption
- USE contraption (`Screwdriver`)
- USE `Screwdriver` on box

## 3. Desert

- USE sand (nn the solar panel)
- PICK UP fuse (`Fuse`)
- LOOK AT `Fuse`
- USE `Screwdriver` on box

## 4. Platform

- USE `Fuse` with contraption (`Fuse`)
- USE `Screwdriver` on box

## 5. Desert

- USE `Screwdriver` on panel (of the robot) (`Panel`, `Screws`)
- USE `Fuse` on access panel
- TALK TO robot
- USE box
  - "'Become the machine gate'"

## 6. Metallurgic Wonders

- USE sign (`Sign`)
- USE `Sign` on `Panel` (`Platform`)
- USE `Screws` on `Platform`
- USE `Platform` on gap
- USE cat
  - "Scratch the cat's chin."
- WALK to the left platform
- PICK UP platform (`Platform`)
- USE button
- USE `Platform` on (left) gap
- WALK left
- PICK UP platform (`Platform`)
- USE `Platform` on (left) gap
- WALK to the left (the jellyfish moves to the right)
- USE cat
  - "Stroke the cat's ears."
- WALK right
- PICK UP jellyfish (it moves to the right)
- PICK UP platform (`Platform`)
- USE `Platform` on (right) gap
- WALK right
- PICK UP jellyfish (it moves to the left)
- PICK UP badge (on the left platform, you know the game with the `Platform`) (`Badge`)
- USE `Badge` on scanner (on the left)
- TALK TO AI
- PICK UP `Platform` (`Platform`)
- USE box
  - "Draw line with finger and say 'To Sarthen'."

## 7. Tower

- USE source control (on the beam in the middle) **2 times** (to power the archive)
- USE archive
  - "Search for most recent entry."
  - "Search for information on Source Gems."
- USE source control (on the beam in the middle) **2 times**
- USE source golem
  - "Command the golem to make s Source Gem."
- USE source golem (`Gem`)
- USE `Gem` on box

## 8. Island

- USE (3rd) stone (in the middle of the island) (note appears)
- USE (3rd) stone **again** (note flies onto the middle pipe)
- USE (2nd) stone
- USE (3rd) stone
- USE (4th) stone **2 times**
- USE (3rd) stone
- PICK UP note (`Note`)
- LOOK AT `Note`
- USE `Note` on box

# Land of Peaks

- USE rock (`Key`)
  - **NOTE:** The `Key` bits with their different lengths and the lamp posts with their different heights
- USE tall lamp
- USE shortest lamp
- USE tallest lamp
- USE short lamp
- USE door
- USE box
  - "Touch the box with both index fingers held together."

## 9. Bridge

- TALK TO Kadin Longblow VII
- LOOK AT board
- TALK TO Kadin
  - "What does that funny board on the ground do?"
- USE `Gem` on board
- USE board
- USE box
  - "'Finth'"

# Finth Tower

- PICK UP **all four** chips (`Chip`, `Chip`, `Chip`, `Chip`)
  - **NOTE:** The chips have a yellow line at different positions
- USE `Chip` with the yellow line at the bottom on the pylon which connects to the lowest part of the pillar (the 4th pylon)
- USE `Chip` with the top yellow marking on 3rd pylon
- USE `Chip` with 2nd to marking on 2nd pylon
- USE `Chip` with 3rd marking on 1st pylon
- USE lift
- USE box
