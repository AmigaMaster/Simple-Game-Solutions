# Secrets Of Sultan Kanuni - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 24.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Archive.org](https://archive.org/details/secrets-of-sultan-kanuni)

## 1. Home

### 1.1. Your Room

- USE door

## 2. Home

- EXIT northwest

## 3. Street to the Park

- PICK UP ATM card (`ATM Card`)
- USE door (to the cafe)

### 3.1. Cafe

- TALK TO waiter
  - **NOTE:** Born 1979
- TALK TO man at the table
- EXIT west, south

## 4. Home

- USE Hortumbank ATM
  - USE `ATM Card` with card slot (top right)
  - ENTER `1979`
  - USE OK (you need to use the ATM Card icon to use the buttons)
  - DRAW MONEY (`Money`)
- EXIT northeast

## 5. Museum Street

- USE (left) door

### 5.1. Electronic Store

- TALK TO shopkeeper
- USE `Money` with shopkeeper (`Vacuum Cleaner`)
- EXIT shop, west **2 times**

## 6. Street to the Park

- EXIT west

## 7. Park

- USE `Vacuum Cleaner` with brown spot
- PICK UP key (`Key`)

## 8. Street to the Park

### 8.1. Cafe

- USE `Key` with man (`Shovel`)

### 8.2. Hardware Store

- TALK TO man
- EXIT west, south, enter your home

## 9. Home

### 9.1. Your Room

- USE computer
  - ID PRINTER (you need to click slightly above the menu entries)
  - INSPECTOR (`Fake Inspector ID`)

## 10. Museum Street

- TALK TO guard
  - "I am on duty..."
- USE `Fake Inspector ID` on guard
- USE door to the museum

### 10.1. Museum Entrance

- TALK TO receptionist
- USE `Money` on receptionist (`Map`, `Coins`)
- EXIT west

## 11. Home

### 11.1. Your Room

- USE computer
  - ANALYZER
  - USE `Map` on screen
  - X-RAY
  - PRINT-YAZDIR (`X-RAY Map`)

## 12. Museum Street

### 12.1. Museum Entrance

- EXIT east

### 12.2. Museum Garden

- WALK east
- USE `Shovel` with spot under the tree
- USE hole
- USE chest **2 times** (`Shoes`, `Document`, `Picture`)
- EXIT west

## 13. Home

### 13.1. Your Room

- USE computer
  - TRANSLATOR
  - USE `Document` on screen
  - MENU
  - ANALYZER
  - USE `Picture` on screen
  - PRINT-YAZDIR (`Picture 2`)
  - QUIT
- PICK UP disk (from the desk) (`Disk`)

## 14. Museum Street

### 14.1. Museum Entrance

- USE yellow book
- USE passage

### 14.2. Passage

- USE `Shoes` on yourself
- USE guard
- PICK UP rifle (`Rifle`)
- EXIT east

### 14.3. Computer

- USE computer
  - WORLD MAP
  - MENU
  - MAGNETIC CARD MANUAL
  - SAVE INFORMATION TO DISK
  - USE `Disk` on disk drive slot (`Disk with Data`)
  - QUIT
- EXIT east

### 14.4. UFO Hall

- USE `Rifle` on alien
- USE door to UFO

#### 14.4.1. UFO

- USE computer
  - MAGNETIC CARD PRINTER
  - USE `Disk with Data` on disk drive slot
  - YAZDIR-PRINT
  - PICK UP magnetic card (`Magnetic Card`)
  - QUIT
- USE door (to exit)

### 14.5. UFO Hall

- USE `Magnetic Card` on security panel

### 14.6. Briefing Room

- USE villains
