# Siren - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 31.01.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Gamejolt](https://gamejolt.com/games/siren/753750)

## 1. Left Side

- USE detritus (on the ground below the left edge of the window under the undergrowth) (`Towel`)
- USE drainpipe
- PICK UP pipe (`Pipe`)
- USE `Pipe` on board (right of the drainpipe)
- USE `Baton` on window
- USE `Towel` on window
- USE window

## 2. House

### 2.1. Fireplace

- USE grate (in the fireplace) (`Stick`)
- USE (right) loose wallpaper (`Wallpaper`)
- LOOK AT lock
- USE `Wallpaper` on door
- USE `Stick` on lock (`Key`)
- USE `Key` on lock

## 3. Right Side

- PICK UP sign (from the door) (`Sign`)
- USE `Sign` on plant pot (`Wet sign`)
- USE `Wet sign` on window
- USE `Flashlight` on window
- USE `Pocket knife` on hinges (of the small hut)

## 4. House

### 4.1. Fireplace with Hole in the Floor

- PICK UP litter (in the middle below of the hole) (`Sandpaper`)
- USE (lower) drawer (`Match`)
- USE `Match` on `Sandpaper` (`Lit match`)
- USE `Lit match` on candle
- PICK UP battery (in the hole)

### 4.2. Upper Floor

- USE (back) door
- USE `Gun` on lock (of back door)
