# Stoaway - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------
|   1.0   | 17.07.2022 | Amiga Master (AmigaMaster-NG@t-online.de)

Get the game for free from [Itch.io](https://georgebroussard.itch.io/stowaway)

## 1. Command Deck

- USE My Equipment Drawer (`Space suit helmet`)
- USE Engineers Equipment Drawer
- PICK UP Welder (`Portable welder`)
- USE `Portable welder` with Inner Hatch
- USE Equipment Drawer to the left (`Fuel canister`)
- USE `Fuel canister` with `Portable welder`
- USE `Portable welder` with Inner Hatch
- USE Command Deck Airlock

## 2. Space

- ENTER Storage Deck

## 3. Storage Deck

- USE Floyd **2 times**
- TALK TO Floyd
- USE Corpse Legs (`Engineer's access card`)

## 4. Command Deck

- USE `Engineer's access card` with Engineers Equipment Drawer
- USE Engineers Equipment Drawer (`Engineer Harley's music card`)

## 5. Space

- Fly around the space ship at the top to the right side
- USE `Engineer's access card` with Auxillery Storage Deck

## 6. Auxillery Storage Deck

- USE Shielding (`Heavy shielding`)
- LOOK AT Notice
- SEARCH through Section C (`Battery`)
- ENTER Storage Chute (Down)

## 7. Storage Deck

- USE `Battery` with Floyd
- USE `Engineer Harley's music card` with Floyd

## 8. Engineering Deck

- USE `Heavy shielding` with Breach
- USE `Portable welder` with Shield
- USE Corpse (`Captain's access card`)
- USE Button

## 9. Command Deck

- USE `Captain's access card` with Captain's Equipment Drawer
- USE Captain's Equipment Drawer (`A Blaster`)
- USE `A Blaster` with Tentacles **3 times**
