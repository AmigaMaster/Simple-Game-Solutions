# Sinkhole Sy - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 17.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/2939-sinkhole-sy)

## 1. Lower Sinkhole

- PULL piece of metal? **2 times** (`Rung`)
- USE `Rung` with (left) old-timey pipe
- LOOK AT smashed pipe (`Clay Shard`)
- USE `Clay Shard` with dirt and rubble (on the ground)
- PUSH metal plate
- PICK UP mess of wiring **2 times** (`Crucial Cable`, `Important Wire`)
- USE `Important Wire` with `Crucial Cable` (`Elongated Wire`)
- USE `Elongated Wire` with something sharp
- PULL wrapped wire (`Elongated Wire`, `Broken Ladder`)
- USE `Elongated Wire` with `Broken Ladder` (`Long Grappling Hook`)
- USE `Long Grappling Hook` with big tree root
- USE long grappling hook

## 2. Upper Sinkhole

- LOOK AT smashed pipe (in the left wall) (`Clay Shard`)
- Climb up
- PICK UP more dirt (on the left side below the gas pipe valve) (`Dirt`)
- USE `Rung` with (left) broken pipe
- USE `Hankie` with drip (a tiny hotspot on the right side below the power generator, it drips a brown liquid from time to time) (`Corrosive Hankie`)
- WALK TO big tree root
  - **NOTE:** USE `Long Grappling Hook` while standing on the big tree root if you had picked it up
- USE long grappling hook

## 3. Lower Sinkhole

- PICK UP long grappling hook (`Long Grappling Hook`)
- USE `Long Grappling Hook` with little tree root
- USE grappling hook
  - USE **both** `Clay Shard` with dinosaur fossil (`Prehistoric Find`)
  - LOOK AT `Prehistoric Find` (`Old Spearhead`)
  - USE grappling hook
- PICK UP grappling hook `Long Grappling Hook`
- USE `Long Grappling Hook` with long tree root
- USE long grappling hook

## 4. Upper Sinkhole

- PICK UP long grappling hook (`Long Grappling Hook`)
- USE ladder
- Climb up
- USE `Prehistoric Find` on (left) pylon
- Climb down
- PICK UP pylon (`Pylon`)
- USE `Corrosive Hankie` with valve (left at the gas line)
- OPEN valve
- CLOSE valve
- OPEN valve
- CLOSE valve
- OPEN valve
- USE `Spearhead` with `Long Grappling Hook` (the gas ignites)
- USE `Corrosive Hankie` with open flame
- WALK TO big tree root
- USE `Long Grappling Hook` on big tree root
- USE long grappling hook

## 5. Lower Sinkhole

- USE `Dirt` with pool of water (`Wet Mud`)
- USE `Wet Mud` with `Pylon` (`Plugged-up Pylon`)
- USE `Plugged-up Pylon` with pool of water
