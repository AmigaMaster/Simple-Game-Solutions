# Deity - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 26.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1287-deity)

## 1. Entrance Hall

- USE (brown) door on the left

## 2. Dining Room

- PICK UP paper (`Diary Entry III`)
- LOOK AT `Diary Entry III`
- USE (left) door

## 3. Kitchen

- PICK UP (brown) broken blade (`Broken Kitchen Knife`)
- LOOK AT blood stain
- **ATTENTION:** SAVE YOUR GAME HERE
- USE (right) door and walk to the Entrance Hall

## 4. Entrance Hall

- **QUICK:**
  - USE (left) brown door and walk to the kitchen

## 5. Kitchen

- USE freezer door

### 5.1. Freezer

- **ATTENTION:** SAVE YOUR GAME HERE - you now have 2 minutes to escape
- PICK UP paper (`Diary VII`)
- LOOK AT `Diary VII`
- LOOK AT cardboard box
- USE cardboard box (to climb on them)
- USE `Broken Kitchen Knife` on freezer motor
- LOOK AT air vent
- USE air vent

## 6. Backyard

- USE (brown) door (to the shed)

### 6.1. Shed

- PICK UP paper (`Diary Entry II`)
- PICK UP screwdriver (`Screwdriver`)
- USE door

## 7. Backyard

- USE `Screwdriver` on (left) door
- USE busted door

## 8. Entrance Hallway

- **ATTENTION:** SAVE YOUR GAME HERE
- USE stairs

## 9. Hallway

- USE (right) door

### 9.1. Study

- PICK UP symbol (`Pentagram Symbol`)
- USE door

## 10. Hallway

- USE (left brown) door

### 10.1. Bedroom

- PICK UP library key (`Library Key`)
- USE door

## 11. Hallway

- USE stairs

## 12. Entrance Hallway

- USE `Library Key` on (right white) door
- USE (right white) door

### 12.1. Library

- PICK UP (dark green) book from the shelf on the back wall (`Book`)
- LOOK AT `Book` (`Silver Key`)
- USE door

## 13. Entrance Hallway

- USE (right brown) door

### 13.1. Lounge

- USE drawers (in the cabinet) (`Symbol`)
- USE door

## 14. Entrance Hallway

- USE (left white) door

### 14.1. Display Room

- USE `Silver Key` on glass case (`Spell Book`)
- USE door

## 15. Kitchen

- USE (left) door

## 16. Backyard

- EXIT east

## 17. Outer Wall

- WALK into manhole

## 18. Sewer

- USE archway

### 18.1. Portal Room

- USE golden wheel
  - USE (purple) `Symbol` on symbol
  - EXIT (lower right corner)
- USE silver wheel
  - USE `Pentagram Symbol` on symbol
  - EXIT
- USE sacred door

### 18.2. Sacred Room

- LOOK AT sacred book
- EXIT south, east, USE ladder, northwest, west

## 19. Kitchen

- USE `Tissue` on broken blade (`Blood Stained Tissue`)

## 20. Sewer

- USE archway

### 20.1. Sacred Room

- USE `Blood Stained Tissue` on sacred water

## 21. Hallway

- USE (left white) door

## 22. Attic

- **ATTENTION:** SAVE YOUR GAME HERE
- USE `Spell Book` on book stand
- **WAIT ans as soon as Robbie is inside the pentagram**
  - USE book
