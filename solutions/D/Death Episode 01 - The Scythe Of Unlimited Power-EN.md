# Death Episode 01: The Scythe Of Unlimited Power - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 12.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1320)

## 1. Home

### 1.1. Bedroom

- PICK UP scythe (`Scythe`)
- PICK UP styrofoam head (`Styrofoam Head`)
- USE `Scythe` on rope

### 1.2. Bathroom

- PICK UP razor (on the floor) (`Razor`)
- PICK UP perfume (in the cabinet) (`Perfume`)
- EXIT west

### 1.3. Bedroom

- EXIT downstairs

### 1.4. Entrance Hall

- TALK TO monkey
- USE box (`Monkeyfood`)
- USE `Monkeyfood` on monkey
- USE `Razor` on `Styrofoam Head` (`Banana Mould`)
- USE `Monkeyfood` on `Paper Bag` (`Bag with Monkeyfood`)
- USE `Perfume` on `Bag with Monkeyfood` (`Bag with Foodmix`)
- USE `Bag with Foodmix` on `Banana Mould` (`Filled Banana Mould`)
- EXIT through the door

## 2. Front Door

- EXIT south

## 3. Chessboard

- EXIT south

## 4. Portal Bridge

- USE `Filled Banana Mould` on torch (`Fake Banana`)
- EXIT south

## 5. Home

### 5.1. Entrance Hall

- USE `Fake Banana` on monkey (`Assignment`)

## 6. Chessboard

- TALK TO Dr. Zombie
- USE `Assignment` on Dr. Zombie (`Portable Teleporter`)

## 7. Portal Bridge

- EXIT north

## 8. Portal

- USE portal

## 9. Bedroom

- USE old lady

## 10. Mouth

- TALK TO tooth rotter
- EXIT north

## 11. Brain

- USE brain
  - PICK UP marmalade
  - PICK UP bowl of sugar
- EXIT south

## 12. Mouth

- PICK UP metal (`Metal`)

## 13. Brain

- EXIT west

## 14. Nose

- USE `Metal` on nosehair (`Electric Metal`)

## 15. Brain

- TALK TO man
- USE `Electric Metal` on brain

## 16. Old People Home

### 16.1. Frontside

- TALK TO guard
- EXIT northeast

### 16.2. Back Alley

- PICK UP skeleton rat (between the containers)
- EXIT east, south

## 17. Home

### 17.1. Frontside

- PICK UP (right) box (`Box`)
- ENTER storage room

#### 17.1.1. Storage Room

- PICK UP rake (`Rake`)
- EXIT east

### 17.2. Frontside

- ENTER front door

#### 17.2.1. Kitchen

- PICK UP cheese (`Cheese`)
- PICK UP frying pan (`Frying Pan`)
- EXIT south **2 times**

## 18. Junkyard

### 18.1. Entrance

- TALK TO guard (`Blueprint`, `Pipe`)
- EXIT north

### 18.2. Junkyard

- USE `Rake` on crane (`Rope`)
- EXIT south **2 times**

## 19. Old People Home

### 19.1. Back Alley

- LOOK AT `Blueprint`
  - EXIT detail view
- USE `Cheese` on `Box` (`A Box with Cheese`)
- USE `Rope` on `Pipe` (`A Rope and a Pipe`)
- USE `A Rope and a Pipe` on `A Box with Cheese` (`A Rat-Trap`)
- USE `A Rat-Trap` on ground
  - USE death (yourself) at the right time to catch the rat
- PICK UP box (`Rat`, `Letter from Emo`)
- USE `Rat` on open window

### 19.2. Frontside

- ENTER door

### 19.3. Bedroom

- PICK UP pink slippers (`Pink Slippers`)
- PICK UP portable teleporter (`Portable Teleporter`)
- USE `Portable Teleporter` on death

## 20. Chessboard

- ENTER tent

### 20.1. Tent

- USE coffin
- ENTER tunnel

#### 20.1.1. Tunnel

- WALK right (until you talk to the Demon Hunter)

### 20.2. Tent

- USE `Pink Slippers` on death
- ENTER tunnel

#### 20.2.1. Tunnel

- PICK UP sword (`Sword`)
- WALK right

### 20.3. Tent

- USE `Pipe` on `Frying Pan`
- ENTER Tunnel

#### 20.3.1. Tunnel

- USE `Sword` on Demon Hunter

### 20.4. Tent

- USE table
- EXIT west

## 21. Chessboard

- USE `Rake` on evil monkey
- USE evil monkey

### 21.1. Tent

#### 21.1.1. Tunnel

- TALK TO Demon Hunter (`Sword`)

### 21.2. Tent

- USE `Sword` on tent roof

## 22. Chessboard

- USE `Rake` on evil monkey
- USE evil monkey

### 22.1. Tent

- USE evil monkey

#### 22.1.1. Tunnel

- WALK to the right but stop when the monkey is next to the Demon Hunter
- TALK TO Demon Hunter
  - "Signal Demonhunter to distract Evilmonkey"
- USE evil monkey (`Key`)

### 22.2. Tent

- USE `Key` on coffin
