# Dead Silence - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 02.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

## 1. House

### 1.1. Study

- USE bookcase (`Witch Craft`)
- LOOK AT `Witch Craft`
- PICK UP box (from desk) (`Box`) (5/5)
- USE door

### 1.2. Living Room

- USE front door
- TALK TO Luke Whitland
- EXIT south

### 1.3. Kitchen

- PICK UP file (`File`) (5/10)
- USE `Box`
  - USE `File` on package
  - PICK UP book (`Science Book`)
- USE living door

### 1.4. Living Room

- USE `Science Book` on Luke (5/15)
- TALK TO Luke
  - "Science"
- USE `Witch Craft` on Luke (5/20)
- TALK TO Luke
  - "Witch Craft"

### 1.5. Kitchen

- USE lab door

### 1.6. Lab

- USE desk (`Ring`) (5/25)
- USE shelf (`Alchemy Book`)
- USE `Alchemy Book`
- PICK UP pry bar (next to the chair) (`Prybar`) (5/30)
- USE kitchen door

### 1.7. Living Room

- USE `Alchemy Book` on Luke (5/35)
- TALK TO Luke
  - "Alchemy"
  - "Enric"
  - "Go to Historian"
  - "Yes"

## 2. Historian House

- USE front door

### 2.1. Living Room

- TALK TO Historian
- USE `Ring` on Historian
- TALK TO Historian
  - "Enric" (`History Book`)
- USE `History Book`
- USE front door

## 3. House

### 3.1. Living Room

- USE `History Book` on Luke (5/40)
- TALK TO Luke
  - "History Book"
  - "Cemetery"
  - "Go to Cemetery"
  - "Yes"

## 4. Cemetery

### 4.1. Entrance

- EXIT east

### 4.2. Crypt

- PICK UP dirt road (`Dirt`) (5/45)
- USE keypad
  - **NOTE:** The inscription
  - ENTER `1974` (The year he entered school) (5/50)
- ENTER entrance

#### 4.2.1. Crypt Chamber

- USE `Prybar` on coffin lid
- USE coffin top (at the right part of the coffin) (`Old Chest`) (5/55)

## 5. House

### 5.1. Lab

- USE `Old Chest` (`Golden Rod`, `Worn Notes`)
- USE `Worn Notes` (5/60)
- USE `Prybar` on `Golden Rod` (`Gold Powder`)

### 5.2. Kitchen

- USE cabinets (`Salt`) (5/65)
- USE `Salt` on `Dirt` (`Salt/Dirt`) (5/70)
- USE `Gold Powder` on `Salt/Dirt` (`Compound`) (5/75)
- USE `Compound` on microwave (`Amerethine`) (5/80)

### 5.3. Lab

- USE `Amerethine` on Enric Halkar (5/85)
- USE `Golden Rod` on Enric Halkar

### 5.4. Living Room

- USE phone
- USE front door
  - "To Historian"

## 6. Historian House

- USE front door to start a cutscene

## 7. House

### 7.1. Kitchen

- USE lab door (5/90)
