# D-List Diva - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|

Download the free game from [itch.io](https://kini-games.itch.io/d-list-diva)

## 1. Theatre

### 1.1. Changing Room

- PICK UP loose thread (on the wardrobe) (`Thread`)
- PICK UP cricket bat (`Cricket Bat`)
- USE sofa (`Sweet`)
- PICK UP eyeliner (`Eyeliner`)
- PICK UP crystal (`Crystal`)
- LOOK AT `Crystal`
- USE `Crystal` on speaker (`Contract`)
- TALK TO Davina
- EXIT north

### 1.2. Bar

- USE `Thread` on pest
- PICK UP catnip toy (`Catnip Toy`)
- USE drinks menu
- TALK TO bartender
  - "Make with the booze, lady."
  - "Tea is fine." (`Tea`)
- USE `Sweet` on `Tea` (`Lemon Tea`)
- USE `Catnip Toy` on `Lemon Tea` (`Herbal Tea with Lemon`)
- PICK UP magazine (`Magazine`)
- EXIT west

## 2. Street

- TALK TO autograph hunter (`Cigarette Paper`)
- USE `Cigarette Paper` on `Contract` (`Modified Contract`)
- USE `Eyeliner` on `Modified Contract` (`Signature`)
- USE `Signature` on autograph hunter (`Tobacco`, `Cigarette Papers`)
- USE `Cigarette Papers` on `Tobacco` (`Unrolled Cigarette`)
- LOOK AT car
- USE `Cricket Bat` on car
- USE `Recording Device` on puddle (`Recording`)

## 3. Theatre

### 3.1. Bar

- TALK TO bartender
  - "Make with the booze, lady."
  - "Make mine a Mojito." (`Mint Leaf`)

### 3.2. Changing Room

- USE `Mint Leaf` on mirror (`Dried Mint Leaf`)
- USE `Dried Mint Leaf` on `Unrolled Cigarette` (`Menthol Cigarette`)
- USE `Menthol Cigarette` on Davina
- USE `Herbal Tea with Lemon` on Davina
- USE `Recording` on Davina
