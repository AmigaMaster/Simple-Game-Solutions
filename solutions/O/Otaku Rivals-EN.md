# Otaku Rivals - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 31.12.2025 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/otaku-rivals)

## 1. Map

- USE biology lab

## 2. Biology Lab

- TALK TO Maleko (`Rope`, `Broom`)
- EXIT south
- LOOK AT **all** colored substances
- TALK TO Maleko
  - "What are those potions?"
- PICK UP green substance (`Green Substance`)
- USE `Green Substance` on microscope
- EXIT south

## 3. Pub

- PICK UP (left) poster (`Hellsing Poster`)
- PICK UP poster (from the strongback) (`"Dark Zero" Poster`)
- TALK TO Haruko
- PICK UP drink (`Drink`)
- USE `Green Substance` on `Drink`
- USE `Drink` on Haruko
- PICK UP Haruko's guitar (`Haruko's Guitar`)
- USE `Rope` on Haruko (`Haruko`)
- EXIT west

## 4. Parking Lot

- TALK TO Kazuma
- USE `Haruko's Guitar` on Kazuma
- USE `Rope` on Kazuma (`Kazuma`)
- PICK UP coin (`Coin`)
- TALK TO Mfu
- EXIT west

## 5. Student Mall

- TALK TO Gutzs
- LOOK AT poster
- PICK UP newsletter (`Newsletter`)
- LOOK AT `Newsletter`
- USE `"Dark Zero" Poster` on poster
  - "...Caskar!"
- TALK TO Gutzs (`Gutzs`)
- TALK TO Ibishop (`Costume`)
- USE `Costume` on Wzzl
- USE `Rope` on Spike (`Spike`)
- EXIT east

## 6. Pub

- TALK TO Shin
- USE `Coin` on denotian can
- TALK TO Shin (`Bottle`)

## 7. Roof

- TALK TO Alucard
- TALK TO Onizuka
- USE `Bottle` on Onizuka
- EXIT east

## 8. Biology Lab

- USE screen
  - USE register
  - EXIT

## 9. Parking Lot

- USE `Rope` on Onizuka (`Onizuka`)
- TALK TO Mfu
  - "I have studied..."
  - "to"
  - "tsu"
- TALK TO Mfu
  - "Are you hiding..." (`Ecci magazine`)

## 10. Mall Back Entrance

- TALK TO N47Q (`Nato's card`)
- LOOK AT `Nato's card`
- TALK TO N47Q
  - "I know a zombie..."
- USE `Hellsing Poster` on N47Q

## 11. Roof

- USE `Broom` on dust (`Alucard's Dust`)

## 12. Student Flats

- WALK TO Griffin's Room

### 12.1. Griffin's Room

## 13. Student Flats
