# Owl's Quest - Every Owl has its day - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 01.01.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Archive.org](https://archive.org/details/owls-quest)

## 1. Cedric's Tree

- USE rock (`Scorpion Trainer`) (1/8)
- EXIT west

## 2. Inverted Tower

- USE inverted tower
- EXIT west

## 3. Desert

- USE `Scorpion Trainer` on scorpion (2/8)
- ENTER tunnel

### 3.1. Tunnel

- PICK UP lamp (`Lamp`) (3/8)
- EXIT east **2 times**

## 4. Inverted Tower

- USE `Lamp` on inverted tower (4/8)
- EXIT east

## 5. Cedric's Tree

- USE hole

### 5.1. Inside Tree

- USE under bed (`Blanket`, `Sock`) (6/8)
- EXIT up

## 6. Cedric's Tree

- USE `Blanket` on wagon
- USE `Sock` on wagon
  - **NOTE:** The hotspot text is missing
- EXIT west (`Snake Slayer/Rock`) (7/8)
- EXIT northeast

## 7. Rocks

- USE `Snake Slayer/Rock` on POOOooisonis snake (8/8)
- EXIT north
