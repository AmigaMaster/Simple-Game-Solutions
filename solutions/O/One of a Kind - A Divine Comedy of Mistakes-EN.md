# One of a Kind: A Divine Comedy of Mistakes - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 01.01.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1222)

## 1. Incubator Room

- PICK UP balloon (`Red balloon`)
- EXIT east

## 2. Sample Room

- EXIT east

## 3. Computer Room

- PICK UP balloon (`Yellow balloon`)
- EXIT west **3 times**

## 4. Workshop

- PICK UP rubber hose (`Rubber hose`)
- PICK UP lots of stuff (`Calliper`, `Glue`, `Machine oil`, `Tiny magnet`, `Wire reel`, `Small mirror`, `Piece of fabric`)
- USE tin can (`Steel ball`)
- PICK UP wooden plank (`Wooden plank`)
- USE closet door
- PICK UP broom (`Broom`)
- PICK UP bellows (`Bellows`)
- EXIT west

## 5. Incubator Room

- LOOK AT cistern
- USE cistern
  - USE `Calliper` on right tube
  - USE `Calliper` on `Wooden plank` (`Marked plank`)
  - EXIT
- EXIT west

## 6. Workshop

- USE `Marked plank` on milling machine (`Wooden plank with hole`, `Wooden circle`)
- USR `Broom` on milling machine (`Broom brush`, `Wooden handle`)
- USE `Wooden circle` on `Glue` (`Wooden circle glued`)
- USE `Wooden circle glued` on `Wooden handle` (`Wooden piston`)
- EXIT west

## 7. Incubator Room

- LOOK AT cistern
- USE cistern
  - USE `Wooden piston` on right tube
  - USE `Wire reel` on `Tiny magnet` (`Tiny magnet on the wire`)
  - USE `Tiny magnet on the wire` on key on the ring (`Calculator key`)
- EXIT west

## 8. Workshop

- EXIT east

## 9. Living Room

- PICK UP photographic picture (`Photographic picture`)
- TALK TO brain
- OPERATE (the leftmost icon in the menu bar) `Red balloon` (`Red inflated balloon`)
- OPERATE `Yellow balloon` (`Yellow inflated balloon`)
- USE `Pen knife` on `Piece of fabric` (`Piece of fabric`)
- USE `Yellow inflated balloon` on `Piece of fabric`
- USE `Red inflated balloon` on `Piece of fabric` (`Flying cloth`)
- USE `Flying cloth` on looking glasses
- USE `Steel ball` on gramophone
- USE `Pen knife` on `Photographic picture` (`Piece of glass`)
- DETACH (the second icon in the menu bar) `Photographic picture` (`Photographic picture`, `Picture frame`)
- USE `Small mirror` on `Picture frame` (`Mirror in the frame`)
- USE `Mirror in the frame` on red beam
- EXIT east

## 10. Library

- PICK UP scroll of parchment (`Scroll of parchment`)
- USE glass tube
  - PICK UP cap
  - USE `Machine oil` on bolt
  - USE `Spanner` on bolt
  - USE `Bellows`  on `Rubber hose` (`Bellows with rubber hose`)
  - USE `Bellows with rubber hose` on glass tube
  - USE bellows
  - USE `Tiny magnet on the wire` on key on the ring (`Calculator key`)
- EXIT west **3 times** ans east **2 times**

## 11. Computer Room

- USE `Scroll of parchment` on plotter
- USE `Calculator key` on calculator keyboard (`Sample search chart`)
- EXIT west

## 12. Sample Room

- USE drum controller
- USE samples drum
