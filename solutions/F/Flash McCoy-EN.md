# Flash McCoy - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 11.06.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Get the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/2620-flash-mccoy)

## 1. Front of Shack

### 1.1. Inside Shack

- PICK UP baseball (from the shelf) (`Baseball`)
- PICK UP books (`Book`)
- LOOK AT `Book` (`Flask of bourbon`)
- USE trunk (`Wrench`)
- USE rug (`Calipers`)
- USE wireless
- USE **all four** drawers (`Pencil`)
- USE photograph (above the desk) (`Hook`)
- EXIT south

## 2. Shack

- USE crate (`Small amount of oil`)
- USE `Baseball` on gas can (`Empty gas can`)
- EXIT west

## 3. Entrance

- PICK UP stick (from the flower bed) (`Cane`)
- USE `Cane` on `Hook` (`Improvised device`)
- USE mailbox (`Package`)
- USE pannier (`Pump`)
- USE `Small amount of oil` on motorcycle (`Full bottle of oil`)
- EXIT east

## 4. Shack

- USE `Improvised device` on barrel (`Hose`)
- USE `Hose` on barrels (`Full gas can`)
- EXIT east

## 5. Airfield

- USE door
- PICK UP map (`Portable Map`)
- USE `Wrench` on propeller
- USE `Full bottle of oil` on engine access
- USE `Pump` on tyre
- USE `Full gas can` on fuel cap

## 6. Shack

### 6.1. Inside Shack

- USE `Calipers` on map
- USE `Pencil` on `Portable Map`

## 7. Airfield

- USE cockpit

## 8. Crash Site

- PICK UP flask (`Empty flask`)
- USE `Empty flask` on cactus
- USE `Calipers` on cactus (`Flask of water`)
- USE `Flask of water` on compass (`Compass`)
- EXIT east

## 9. Eastern Crash Site

- EXIT east

## 10. Viewpoint

- USE corpse (`Key`)
- EXIT southwest

## 11. Eastern Crash Site

- USE `Key` on box (on the right side) (`Knife`)
- USE `Knife` on bundle (under the plane on the left wing) (`Parachute`)

## 12. Viewpoint

- USE `Parachute` on tree
