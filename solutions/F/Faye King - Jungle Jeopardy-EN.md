# Faye King: Jungle Jeopardy - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 07.06.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Get the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1871-faye-king-jungle-jeopardy)

## 1. Crash Site

- PICK UP pile of branches (on the right side beside the tree on the ground) (`Pile o branches`)
- EXIT south

## 2. Jungle Path

- USE parrot (it flies away)
- USE `Penknife` on vine (hanging on the left above the rock) (`Jungle vine`)
- EXIT west

## 3. Flower Path

- EXIT northwest

## 4. Tire Path

- TALK TO (left) tourist **2 times**
- EXIT south

## 5. Shop

- USE rubbish bin (`Spray can`)
- LOOK AT litter **2 times**
- PICK UP litter (`Credit note`)
- LOOK AT `Credit note`
- TALK TO Native
- LOOK AT **both** notices beside the door
- ENTER shop

### 5.1. Inside Shop

- USE wooden carving (in the shelf)
- USE `Credit note` on shop keeper
- PICK UP wooden carving (`Wooden carving`)
- EXIT west, north

## 6. Tire Path

- TALK TO (left) tourist
- USE `Wooden carving` on (left) tourist (`Translation book`)

## 7. Shop

- USE `Translation book` on Native
- EXIT north **2 times**

## 8. Rock on Tree Path

- **ATTENTION:** SAVE YOUR GAME HERE
- EXIT east

## 9. Path with Rope

- PICK UP stick (`Nobbly, twiggy stick`)
- LOOK AT rope
  - **ATTENTION:** DO NOT STEP ONTO THE ROPE
- USE `Nobbly, twiggy stick` on rope
- EXIT north

## 10. Tree Stump

- EXIT west

## 11. Waterfall

- PICK UP fishing line (on a stone near the river) (`Fishing line`)
- USE `Jungle vine` on rock face
- USE jungle vine ladder

### 11.1. Cave

- TALK TO Jungle Bob (`Bob's repellent mixture`)
- EXIT north, east

## 12. Tree Stump

- USE `Bob's repellent mixture` on `Spray can`
- USE `Spray can` on swarm
- EXIT east

## 13. Camp

- PICK UP book (`Empty jar`)
- EXIT west, south, west, south **2 times**

## 14. Shop

### 14.1. Inside Shop

- PICK UP nets
- USE `Coins` on shop keeper
- PICK UP nets (`Net`)
- EXIT west, north, east

## 15. Flower Path

- USE `Net` on butterfly (`Butterfly in a jar`)
- EXIT northwest

## 16. Tire Path

- USE `Net` on bug (`Bug`)
- EXIT north, east, north, east

## 17. Camp

- USE `Butterfly in a jar` on biologist
- PICK UP book (`Plant encyclopedia`)
- EXIT west **2 times**

## 18. Waterfall

- USE `Penknife` on `Nobbly, twiggy stick` (`Stick`)
- USE `Fishing line` on `Stick` (`Fishing Pole`)
- USE `Bug` on `Fishing Pole` (`Baited fishing pole`)
- USE `Baited fishing pole` on river **3 times** (`Red Herring`, `Silver Perch`)
- EXIT east, south, west **2 times**

## 19. Jungle Corner

- **ATTENTION:** SAVE YOUR GAME HERE
- USE `Silver Perch` on tiger
- USE `Plant encyclopedia` on flower
- USE flower
- EXIT northwest

## 20. Mouse Burrow

- USE `Pile of branches` on burrow
- USE `Lighter` on pile of branches
- USE `Net` on rodent (`Rodent`)
- EXIT south

## 21. Jungle Corner

- USE `Rodent` on flower (`Venus' pride`)
- EXIT east **2 times**, north, west, up into the cave

## 22. Waterfall

### 22.1. Cave

- USE `Venus' pride` on Jungle Bob (`Bob's special remedy`)
- EXIT north, east, south, west, south, east **3 times**

## 23. Bridge

- **ATTENTION:** SAVE YOUR GAME HERE
- LOOK AT Tiki
- EXIT west, north

## 24. Crash Site

- PICK UP mask (`Mask`)
- LOOK AT `Mask`
- EXIT south, west, northwest, south

## 25. Shop

- USE `Mask` on Native
- USE `Translation book` on Native
- EXIT north, east **3 times**

## 26. Bridge

- TALK TO Native
- EXIT east

## 27. Cave Entrance

- ENTER cave

### 27.1. Cave

- **ATTENTION:** SAVE YOUR GAME HERE
- **NOTE:** TO scroll through the inventory, use the left arrow to scroll to the latest inventory items
- USE `Penknife` on trip wire (`Broken penknife`)
- PICK UP cut trip wire (the left part) (`Trip wire`)
- USE metal container **2 times** (`Dynamite`)
- EXIT west

### 27.2. Cave Camp

- **ATTENTION:** SAVE YOUR GAME HERE
- PICK UP rock (right next to you) (`Rock`)
- USE `Rock` on Native
- PICK UP fire (`Burning log`)
- USE `Burning log` on rope
- USE `Bob's special remedy` on Faye's passenger
- USE `Dynamite` on rock pile
- USE `Trip wire` on dynamite
- USE `Lighter` on fuse wire
- EXIT north

### 27.3. Cave Passage

- EXIT west

## 28. Clearing

- USE vehicle
