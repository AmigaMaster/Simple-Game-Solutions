# Earl Bobby is Looking for a Loo - Walkthrough

## 1. Street

- EXIT north

## 2. Toilet

- USE door
- USE small heart
  - "Bye"
- USE grass (beside the toilet)
- USE AT hole
  - PICK UP knife (`Shepherds Knife`)
- PICK UP hosepipe (`Hosepipe`)
- LOOK AT pipe (he grey one in the grass on the right)
- EXIT southwest

## 3. Street

- USE `Key` on engine bonnet
  - PICK UP canister (`Canister (Empty)`)
  - USE bonnet
- USE tank nozzle
- USE `Hosepipe` on tank nozzle
- USE `Canister (Empty)` on hosepipe
- USE hosepipe
- PICK UP canister (`Canister (Petrol)`)
- TALK TO Baron Mucki
  - "sausage" (`Mucki's Sausage`)
- PICK UP toadstool (in the grass on the right stone) (`Mushroom`)
- EXIT west

## 4. Sheep

- USE `Mushroom` on `Mucki's Sausage` (`Sausage (Poisoned)`)
- USE `Sausage (Poisoned)` on watchdog
- USE `Shepherds Knife` on black sheep (`Sheep's Wool`)
- USE `Shepherds Knife` on branch (`Branch`)
- EXIT west

## 5. Street

- USE `Sheep's Wool` on rock
- USE `Branch` on wool
- PICK UP smouldering wool (`Burning Sheep's Wool`)
- PICK UP Arnold's hand (in the back of the car) (`Arnold's Hand`)
- EXIT north

## 6. Toilet

- USE `Arnold's Hand` on cap (of the pipe)
- USE cap
- USE `Canister (Petrol)` on pipe
- USE `Burning Sheep's Wool` on pipe
- USE loo
