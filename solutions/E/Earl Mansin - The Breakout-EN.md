# Earl Mansin: The Breakout - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 03.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Get the free game from [archive.org](https://archive.org/details/Earl_Mansin)

## 1. Cell

- LOOK AT clock
- PICK UP food (`Food`)
- LOOK AT `Food` (`Fork`)
- PICK UP soap (`Soap`)
- USE `Soap` on `Fork` (`Oiled Fork`)
- USE `Oiled Fork` on loose brick
- PICK UP hacksaw (`Hacksaw`)
- USE `Hacksaw` on barred window

## 2. Office

- USE desk
  - PICK UP green cloth
  - PICK UP book
  - PICK UP receipt
  - PICK UP key (`Small Silver Key`)
- LOOK AT (left middle) drawer (in the file cabinets on the left)
- USE `Small Silver Key` on (left middle) drawer
- USE open drawer
  - PICK UP paperclip (`Paperclip`)
  - EXIT
- USE `Paperclip` on keyhole (door on the far right)
- USE door

## 3. Hallway

- **ATTENTION:** SAVE YOUR GAME HERE
- USE circuit box
- USE circuit breaker
- TALK TO (green) wire (in the corner)
- USE guard (`Keys`)
- USE `Keys` on (third) door
- USE (third) door

### 3.1. Storage Room

- PICK UP chainsaw (ìt does not work)
- PICK UP bicycle tube (around the chainsaw) (`Tube`)
- PICK UP brick (`Brick`)
- USE `Brick` glass
- PICK UP hacksaw (`Hacksaw`)
- PICK UP brick (in the cabinet)
- EXIT east

## 4. Hallway

- USE (left) door

## 5. Office

- USE `Hacksaw` on cupboard closet
- PICK UP library card (`Library Card`)
- PICK UP chloroform (`Chloroform`)

## 6. Hallway

- USE `Library Card` on (second) door
- USE (second) door

### 6.1. Laundry Room

- PICK UP bed sheets (`Bed sheet`)
- PICK UP scissors (`Scissors`)
- PICK UP hammer (`Hammer`)
- USE `Scissors` on `Bed sheet` (`Rag`)

## 7. Office

- USE `Hacksaw` on barred window
- USE `Bed sheet` un-barred window

## 8. Hallway

### 8.1. Laundry Room

- PICK UP bed sheets (`Bed sheets`)
- USE `Bed sheet` on `Bed sheets` (`Rope`)

## 9. Hallway

- USE `Keys` on (right) door
- USE (right) door

### 9.1. Control Room

- LOOK AT guard
- There are two ways to knock out the guard:
  1. Chloroform
     - USE `Rag` on `Chloroform` (`Chloroform Rag`)
     - USE `Chloroform Rag` on guard
  2. By Force
     - USE `Hammer` on guard **2 times**
- PICK UP book (`Book`)
- LOOK AT `Book` (`Article`)
- LOOK AT `Article`
- EXIT west

## 10. Office

- USE `Rope` on un-barred window

## 11. Prison Yard

- PICK UP sign (`Sign`)
- USE `Hammer` on `Sign` (`Board`, `Pole`, `Nails`)
- USE `Hacksaw` on `Board` (`Planks`)
- USE `Pole` on `Planks` (`Y shaped Construct`)
- USE `Tube` on `Y shaped Construct` (`Slingshot`)
- USE `Slingshot` on hole
- USE `Brick` on slingshot
- WALK right
- USE crack
