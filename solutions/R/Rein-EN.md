# Rein - Walkthrough

Get the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1319)

## 1. Lab

- **QUICK:**
  - OPEN (left) door

## 2. Hallway

- TALK TO Henry (`Key`)
- **QUICK:**
  - USE (left) door

## 3. Decontamination Room

- PICK UP (long) metal pipe (`Metal Pipe`)
- USE `Metal Pipe` with metal slab
- PICK UP sledge (`Sledge`)
- USE `Sledge` with pipe on slab
- PICK UP sharpened pipe (`Sharpened Metal Pipe`)
- PICK UP metal slab (`Metal Slab`)
- USE `Metal Slab` with broken door
- USE `Sharpened Metal Pipe` with broken door
- USE broken door

## 4. Pathway

- WALK onto the tiles with the least cracks. From the starting point:
  - left **2 times**, down, left **2 times**, up **2 times**, left **2 times**
- USE (left) door

## 5. Hub 10

- PICK UP handgun (`Handgun`)
- USE `Handgun` with padlock
- OPEN emergency exit
- USE surface ladder
