# A G-G-Ghost! - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 27.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/a-g-g-ghost)

## 1. Street

- ENTER ghost manor

## 2. Ghost Manor

### 2.1. Entrance Hall

- EXIT upstairs

### 2.2. Second Floor Staircase

- EXIT northeast

### 2.3. Second Floor Hallway

- USE lock
- EXIT east

### 2.4. Second Floor Staircase

- USE door

#### 2.4.1. Toilet

- PICK UP spooky soap (`Spooky Soap`)
- USE door

### 2.5. Second Floor Staircase

- EXIT downstairs

### 2.6. Entrance Hall

- EXIT south

### 2.7. Corridor

- USE (right) door

#### 2.7.1. Kitchen

- USE fridge door
- PICK UP banana peel (`Banana Peel`)
- PICK UP jar o' mayonnaise (`Mayonnaise`)
- USE freezer door
- **ATTENTION:** SAVE YOUR GAME HERE
- USE door to EXIT kitchen

### 2.8. Corridor

- **QUICK:**
  - USE `Banana Peel` on floor
- PICK UP dagger (`Dagger`)
- USE door on the back wall

#### 2.8.1. Empty Room

- USE `Dagger` on can of paint (`"Bloody" Dagger`)
- USE door to EXIT room

### 2.9. Corridor

- EXIT west

### 2.10. Entrance Hall

- EXIT east

#### 2.10.1. Nursery

- PICK UP green marker (`Green Marker`)
- USE `Green Marker` on `Mayonnaise` (`Jar of "Ectoplasm"`)
- USE fairy doll
- USE `"Bloody" Dagger` on `Spooky Soap` (`Crude Soap Figurine`)
- USE `Crude Soap Figurine` on fairy doll (`Fairy Doll`)
- USE door to EXIT room

### 2.11. Kitchen

- USE `Fairy Doll`on blender
- USE blender
- PICK UP fairy dust (`Fairy Dust`)

### 2.12. Second Floor Hallway

- USE `Fairy Dust` on cloaked stranger
- USE `Jar of "Ectoplasm"` on cloaked stranger
- USE `"Bloody" Dagger` on cloaked stranger (`Key`)
- USE `Key` on lock
- USE the path to destiny

### 2.13. Path to Destiny

- WALK east

### 2.14. Laboratory

- PICK UP power cable #1 (`Power Cable #1`)
- USE `Power Cable #1` on doohickey
- PICK UP power cable #2 (`Power Cable #2`)
- USE `Power Cable #2` on Nikola Tesla
- USE control panel
