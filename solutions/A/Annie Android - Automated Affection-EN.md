# Annie Android: Automated Affection - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 30.06.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/annie-android-automated-affection)

## 1. Home Street

### 1.1. Home

- USE controls (in the lower left corner)
- USE compartment (`Seed`)
- USE tech support unit (in the shelf on the right) (`Tech Support Device`)
- USE door

## 2. Home Street

- USE `Seed` on soil
- USE fruit (`Waterfruit`)
- TALK TO Mailbot
- LOOK AT Mailbot
  - **NOTE:** Serial `24113`
- USE `Tech Support Device` on Mailbot
  - SELECT 3
    - ENTER serial `24113`
- TALK TO Mailbot (`Message`)
- LOOK AT `Message`
- TALK TO Mailbot
- USE door

## 3. Home Street

- USE `Message` on Messagebot
- USE door

## 4. Home Street

- EXIT east

## 5. Eastern Street

- EXIT east

## 6. Weather Module

- TALK TO Deathbot
  - "What sort of robot are you?"
  - "You're pretty cute..."
  - "Yes, but look at you!..."
  - "Awww, you're so adorable..."
- USE CHAOS HAMMER 7000! (`CHAOS HAMMER 7000!`)
- EXIT south, west **2 times**

## 7. Park

- TALK TO Parkbot (`Key`)
- EXIT west

## 8. RoboHQ

- USE `Key` on speaker
- USE `CHAOS HAMMER 7000!` on window
- USE glass shard (`Shard of Glass`)
- USW window (`Code Breaker`)
- EXIT east

## 9. Park

- TALK TO Parkbot
- USE `Code Breaker` on Birdbot (`Virus`)
- EXIT east **2 times**

## 10. Eastern Street

- WALK to the left behind the robot
- USE button (a hotspot which is visible on the robot's chest when it swings its hammer)
- LOOK AT (right) Assassinbot
  - **NOTE:** Serial number `14223`
- USE `Tech SUpport Device` on (right) Assassinbot
  - SELECT 3
  - ENTER `14223`
- EXIT east

## 11. Weather Module

- USE door (so the Guardbot moves his arm)
- USE `Shard of Glass` on Guardbot
- USE door

### 11.1. Inside Weather Module

- USE controls
- USE `Waterfruit` on opening
- USE `Virus` on opening
