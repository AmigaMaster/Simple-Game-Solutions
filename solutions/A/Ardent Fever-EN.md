# Ardent Fever - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 30.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://archive.org/details/ardent-fever)

## 1. Brig

- IMPRINT (right-click to change the cursor to phlebotinum) moist area (`Moisture`)
- EXIT upstairs

## 2. Hallway

- USE closet door
- IMPRINT broom (`Broom`)
- IMPRINT toolbox (`Scissors`, `Screw Driver`)
- USE stairs to go up
- ENTER Ecological Room

### 2.1. Ecological Room

- IMPRINT animal tranquilizers (`Tranquilizers`)
- USE `Broom` on very large cat
- IMPRINT cherry blossom (`Flowers`)
- EXIT room

## 3. Hallway

- ENTER laboratory

### 3.1. Laboratory

- TALK TO Dr. Delicti
- IMPRINT volatile flask (on the table, the flask with a light red bottom) (`Volatile Chemical`)
- USE `Volatile Chemical` on curtain
- TALK TO captain
- USE `Flowers` on captain
- USE `Scissors` on intravenous fluid transfer
- USE `Tranquilizers` on intravenous fluid transfer
- TALK TO captain
- USE touch screen computer
- EXIT south

## 4. Hallway

- EXIT east

## 5. Sleeping Quarters

- EXIT east

## 6. Control Room

- USE C3DO (teh robot on the view port) (`Warm Cup of Coffee`)
- USE computer controls
- EXIT south, west

## 7. Hallway

- EXIT west

## 8. Engine Room

- TALK TO Eddie the Engineer
- USE `Moisture` on hot processor - **NOTE:** There is no hotspot name
- TALK TO Eddie the Engineer
- USE `Warm Cup of Coffee` on door below the engine  - **NOTE:** There is no hotspot name
- TALK TO Eddie the Engineer
- EXIT east

## 9. Hallway

- WALK to the upper deck
- USE `Screw Driver` on (round) panel
- USE wires
- EXIT east

## 10. Sleeping Quarters

- TALK TO Third Mate Arden
