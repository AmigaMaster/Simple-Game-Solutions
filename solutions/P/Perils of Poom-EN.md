# Perils of Poom - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 26.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/perils-of-poom)

## 1. Locker Room

- USE top right locker **2 times**

## 2. Storage Room

- USE coin return lever
- USE return slot (`Coin`)
- USE `Coin` on vent panel
- PICK UP shiny object (`Small Key`)
- USE `Small Key` on panel (of the soda machine)
- USE opening

## 3. Break Room

- USE `Coin` on man

## 4. Elevator Hallway

- USE call button (left of the elevator)

## 5. Apartment

- USE bed
- USE door

## 6. Map

- TRAVEL to the location at the top of the screen.

## 18. Spaceship

- "Set course for Tooteo!"

## 19. Service Station

### 19.1. Waste Chute

- EXIT east **2 times**

### 19.2. Hallway to Toilets

- ENTER doorway

### 19.3. Cafeteria

- USE buttons (on the soda machine) (`Coins`)
- EXIT west **2 times**

### 19.4. Souvenirs Hallway

- ENTER doorway

#### 19.4.1. Souvenir Shop

- PICK UP novelty gas mask (on the right side) (`Gas Mask`)
- PICK UP poster (inside the tube, the leftmost roll) (`Poster`)
- PICK UP magnet (from the counter) (`Fish Magnet`)
- EXIT south

### 19.5. Souvenirs Hallway

- USE `Fish Magnet` on paint can (`Painted Fish Magnet`)

#### 19.5.1. Souvenir Shop

- USE `Painted Fish Magnet` on fish (`Fish`)

### 19.6. Cafeteria

- PICK UP drink (from the table) (`Drink`)
- USE `Fish` on `Drink` (`Fishy Drink`)
- USE `Fishy Drink` on shifty looking guy
- USE shifty looking guy (`Trading Card`)
- WALK east until someone calls you
- USE (right) pillar

#### 19.6.1. Behind Pillar

- TALK TO kid
- USE `Trading Card` on kid (`Stink Bomb`)
- EXIT west **2 times**

### 19.7. Hallway to Toilets

- USE `Poster` on vent
- USE `Stink Bomb` on queue
- EXIT east

### 19.8. Garbage Bin

- PICK UP wire (on the left) (`Wires`)
- PICK UP metal (`Hyperdrive`, `Metal`)
- PICK UP discard gun (`Flare Gun (Not Loaded)`)
- USE `Wires` on `Metal` (`Grappling Hook`)
- USE `Grappling Hook` on `Flare Gun (Not Loaded)` (`Flare Gun`)
- USE `Flare Gun` on opening (on the upper right screen)

#### 20.5.1. Outside

### 20.6. Map

- TRAVEL to the center location

### 20.7. Film Set

- USE `Empty Bottle` on alcohol (`Alcohol`)

### 20.8. Tree

- TALK TO naturists
- PICK UP shovel (you cannot)
- USE `Lens` on  naturists

### 20.9. Bar

#### 20.9.1. Inside Bar

- USE door (on the right)

##### 20.9.1.1. Kitchen

- EXIT west

- USE `Dig Master 3000` on familiar faces

### 20.10. Desert

- USE `Shaken Alcohol` with sand
- RETURN here (bottom center of the map)
- PICK UP rope (`Rope`)
- USE `Seeds` on hole
- EXIT east

### 20.11. Map

- TRAVEL to obstruction on the lower left

### 20.12. Obstruction

- USE rope on pointy rock
- USE rope

- You are attacked by a worm and avoid being smashed **10 times**. The worm attacks at the following locations:
  1. Top
  2. Bottom
  3. Top
  4. Middle
  5. Top
  6. Middle
  7. Middle
  8. Bottom
  9. Top
  10. Top (`Jar`)
