# Phoenix1 - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 29.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1323-phoenix1)

## 1. Brain Level

### 1.1. Cross Walk

- EXIT northeast

### 1.2. North East Path

- USE 3 hive cells

### 1.3. Cross Walk

- USE light rod

## 2. Living Level

### 2.1. Cross Walk

- EXIT southwest

### 2.2. South West Path

- USE door (at the end of the path)

### 2.3. Ring Walkway

- USE button
- ENTER path

#### 2.3.1. Room 203

- USE `Reset Keycard` on error text (on the wall)
- USE menu (on the right side) - **NOTE:** There is no hotspot text
  - USE living room
  - EXIT menu
- USE (left) screen
  - USE U.M.S Mission Briefing
  - EXIT menu
- USE (right) menu
  - USE Bedroom
- USE shelves
  - USE bottom middle shelf (which looks like there are books in it)
  - PICK UP keycard (`AI-Disable Keycard`)
  - EXIT shelves
- USE (right) menu
  - USE EXIT Room

### 2.4. Ring Walkway

- USE `AI-Disable Keycard` on liquid wall
- EXIT east (trough the doorway) and return to the center

### 2.5. Cross Walk

- USE light rod

## 3. Brain Level

### 3.1. Cross Walk

- EXIT southeast

### 3.2. Brain Room

- USE `Reset Keycard` on liquid wall
- EXIT east

### 3.3. Cross Walk

- USE light rod
  - USE Forest Level

## 4. Forest Level

### 4.1. Light Wall

- EXIT south

### 4.2. Forest Path

- EXIT north

### 4.3. Forest Bridge

- EXIT east

### 4.4. Liquid Wall

- USE liquid wall
- USE "press here" text (`Glove`)
- EXIT south, west, south

### 4.5. Light Wall

- USE light wall

## 5. Cockpit Level

- USE `Glove` on light floor
  - USE `Glove` on liquid wall
    - USE ship data
    - USE navigations
  - USE `Glove` on light circle

## 6. Earth 2.0

### 6.1. Statue

- EXIT east

### 6.2. Graffiti

- LOOK AT graffiti
  - **NOTE:** `3112`
  - EXIT graffiti
- USE console (in the middle of the columns)
  - USE console
  - EXIT console
- EXIT west **2 times**

### 6.3. Wall

- USE (left back) console (the one, that is not broken)
  - USE console
  - EXIT console
- USE (right, front) console
  - USE console
  - EXIT console
- EXIT west
