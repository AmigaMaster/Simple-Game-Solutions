# Yip Quest - Walkthrough

| Version | Datum      | Autor                                     |
|:-------:|------------|-------------------------------------------|
|   1.0   | 29.11.2025 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Itch.io](https://ramoramainteractive.itch.io/yip-quest)

## 1. Hall

> Raga

- LOOK AT portrait
- EXIT east
- SWITCH TO Ram

> RAM

- USE armor
- SWITCH TO Kog

> Kog

- USE gap

### 1.1. Kitchen

- USE barrel (`apples`)
- PICK UP wooden spoon (`wooden spoon`)
- PICK UP knife (`knife`)
- PICK UP mutton (`mutton`)
- USE gap

## 2. Hall

- ENTER dungeon

### 2.1. Dungeon

- TALK TO prisoner
- USE `mutton` on prisoner
- PICK UP violin with bow (`Violine with bow`)
- PICK UP hay (`hay`)
- USE `hay` on bowl (`blonde wig`)

## 3. Hall

- USE `knife` on column (`sharp knife`)
- USE `sharp knife` on curtain (`scrap`)
- USE `sharp knife` on `wooden spoon` (`cut spoon`)
`sharp knife` on armor (`feathers`)

### 3.1. Dungeon

- USE `scrap` on prisoner
- USE `cut spoon` on bowl (`gooed spoon`)
- USE `gooed spoon` on `feathers` (`spoon arrow`)
- GIVE `spoon arrow` to Raga
- GIVE `Violine with bow` to Raga
- SWITCH TO Raga

> Raga

- LOOK AT hay (`needle`)
- USE skeleton (on the ground) (`thread`)
- USE `needle` on prisoner
- USE `thread` on prisoner (`dress`)
- USE `spoon arrow` on skeleton (on the wall)
- PICK UP dunce cap (`dunce cap`)
- GIVE `dunce cap` to Kog
- SWITCH TO Kog

> Kog

### 3.2. Kitchen

- USE `dunce cap` on pot
