# The Devil's Teeth - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------
|   1.0   | 19.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Gamejolt](https://gamejolt.com/games/Devilsteeth/783976)

## 1. House

### 1.1. Living Room

- USE (left) cabinet
  - PICK UP **all** bottles (`Brandy`, `Rum`, `Gin`, `Scotch`, `Vodka`)
  - PICK UP matchbox (`Matches`)
  - BACK
- USE books (above the cabinet)
  - USE Edward Lear
    - FLIP to the last page
    - **NOTES:**
      1. `Bronte's apotheosis reveals what lies within.`
      2. Lord Nelson being created the Duke of Bronté in 1805 (the year Nelson died at `Trafalgar`) - this is his "apotheosis"
  - USE Great Sea Battles
    - READ Trafalgar `1805`
  - USE International Signal Code
    - **NOTE:**
      1. Flag "K": I wish to communicate with you (Yellow/Blue = Green)
      2. Flag "L": Stop instantly (Yellow / Black)
      3. Flag "U": Running into danger (Red/White)
- USE chair (`Pennies`)
- PICK UP syrup tin (in the shelf at the back wall) (`Small key`)
- PICK UP salt cellar (`Salt`)
- USE `Small Key` on drawer (in the table) (`Scissors`)
- USE cabinet (beside the bed) (`Bedsheet`, `Pillow Case`)
- EXIT south

## 2. Outside

- USE crate (`Paintbrush`)
- ENTER house

## 3. House

### 3.1. Living Room

- USE trap door

### 3.2. Basement

- USE tool box (`Snips`, `Screwdriver`)
- USE trunk (`String`)
- PICK UP shell (the yellow thing at the floor on the shelf) (`Shell`)
- USE `Screwdriver` on paint tin (behind the shell)
- USE red box (`Empty Rocket`)
- USE blue box (`Needle and Thread`)
- USE `Shell` on vice (`Gunpowder`)
- USE `Gunpowder` on `Empty Rocket` (`Incomplete Rocket`)
- USE `Incomplete Rocket` on `Salt` (Natrium = Yellow color) (`Incomplete Rocket`)
- USE `Pennies` on `Snips` (`Copper Shards`)
- USE `Copper Shards` (Blue color) on `Incomplete Rocket` (`Almost complete Rocket`)
- USE `String` on `Rum` (`Flammable String`)
- USE `Flammable String` on `Almost complete Rocket` (`Complete Rocket`)
- USE `Paintbrush` on (white) tin (in the shelf)
- USE `Paintbrush` on paint tin
- USE `Paintbrush` on `Bedsheet` (`Improvised Flag (Red/White)`)
- USE sea chest
  - Set combination `1805`
  - USE catch (`Souwester`)
- USE `Scissors` on `Souwester` (`Yellow Squares`)
- USE `Pillow Case` on coal store (`Black pillowcase`)
- USE `Scissors` on `Black pillowcase` (`Black Squares`)
- USE `Black Squares` on `Yellow Squares` (`Improvised Flag (Black/Yellow)`)
- USE ladder

## 4. Outside

- USE `Complete Rocket` on bottle (next to the flagpole)
- USE `Matches` on Rocket
- USE `Improvised Flag (Black/Yellow)` on flagpole
- USE `Improvised Flag (Red/White)` on flagpole
