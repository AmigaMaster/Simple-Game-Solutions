# The Last Bus - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 23.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/2753)

## 1. Main Street

- TALK TO old man **2 times**
- EXIT northwest

## 2. Alley

- ENTER "Dough or Die"

### 2.1. Dough or Die

- PICK UP glazed donut (you cannot)
- EXIT shop

## 3. Alley

- TALK TO waitress

### 3.1. Dough or Die

- PICK UP glazed donut (you cannot)

## 4. Main Street

- PICK UP takeout bag (`Takeout Bag`)

## 5. Alley

### 5.1. Dough or Die

- USE `Takeout Bag` on glazed donut (`Glazed Donut`)
- PICK UP pie server (`Pie Server`)

## 6. Main Street

- USE `Pie Server` on loose gravel (`Pie Server with Gravel`)
- USE `Pie Server with Gravel` on `Glazed Donut` (`"Chocolate Donut"`)

## 7. Alley

- USE `"Chocolate Donut"` on waitress
- PICK UP bus keys (`Bus Keys`)

## 8. Main Street

- USE `Bus Keys` on parked bus
