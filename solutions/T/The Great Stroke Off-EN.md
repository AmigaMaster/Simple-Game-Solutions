# The Great Stroke Off - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 15.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/540-great-stroke-off-the)

## 1. Crosswalks

- INTERACT WITH door

### 1.1. Stickz & Ballz

- TALK TO uncle Ernie **2 times**
  - "Uncle Ernie, can you demonstrate..."
- **QUICK:**
  - PICK UP hot dogs (`Hot Dog`)
- TALK TO uncle Ernie
  - "Ahhh, yes, can you read rule six for me?"
- **QUICK:**
  - PICK UP key (`Big Key`)
- EXIT east

## 2. Crosswalks

- EXIT east

## 3. Toilet

- INTERACT WITH `Big Key` on toolshed
- INTERACT WITH toolshed

### 3.1. Toolshed

- PICK UP screwdriver (from the toolbox on the left) (`Screwdriver`)
- PICK UP fishing net (`Fishing Net`)
- PICK UP chainsaw (`Chainsaw`)
- EXIT south

## 4. Toilet

- INTERACT WITH toilet door

### 4.1. Inside Toilet

- LOOK AT toilet
- USE `Fishing Net` with toilet (`Yellow Ball`)
- INTERACT WITH door

## 5. Toilet

- EXIT west

## 6. Crosswalks

- EXIT northeast

## 7. Mini-Golf Course

### 7.1. Killer Klown

- LOOK AT trash bin
- INTERACT WITH trash bin (`Blue Ball`)
- EXIT west

## 8. Crosswalks

- EXIT north

## 9. Arkade

- INTERACT WITH arcade door

### 9.1. Arcade Entrance

- EAT `Hot Dog` (`Half a Hot Dog`)
- USE `Half a Hot Dog` with small hole (mouse hole)
- EXIT east

### 9.2. Arcade Back Room

- INTERACT WITH drunken violent bums (one of the machines)
- EXIT west

### 9.3. Arcade Entrance

- PICK UP small hole (`Quarter`)
- USE `Quarter` with coin slot (in the Point 'n Shoot machine)
- PICK UP white ball (`White Ball`)
- INTERACT WITH door (to leave)

## 10. Crosswalks

- EXIT north

## 11. Haunted House

- INTERACT WITH door

### 11.1. Mirror Room

- LOOK AT odd mirror (next to the door)
- USE `Screwdriver` with odd mirror
- PICK UP odd mirror (`Mirror`)
- INTERACT WITH ladder

### 11.2. Attic

- LOOK AT gears (in the ghost lifter machine)
- INTERACT WITH lever (turn the machine off)
- USE `Screwdriver` with gears (`Red Ball`)
- INTERACT WITH lever (turn the machine on)
- INTERACT WITH ladder

### 11.3. Mirror Room

- WALK west
- INTERACT WITH ghost (`Hockey Mask`)
- EXIT west

### 11.4. Backside

- EXIT east **2 times**

## 12. Crosswalks

### 12.1. Stickz & Ballz

- USE `Red Ball` with ball washer
- USE `White Ball` with ball washer
- USE `Blue Ball` with ball washer
- USE `Yellow Ball` with ball washer
- INTERACT WITH ball washer

## 13. Mini-Golf Course

- **NOTE:** SAVE YOU GAME HERE

  1. Nerds
  2. Punks

## 14. Nerds

### 14.1. Toilet

- USE `Chainsaw` with old tree
- USE `Chainsaw` with fallen (`Stump`)

#### 14.1.1. Toolshed

- PICK UP batteries (`Batteries`)

### 14.2. Mini-Golf Course

#### 14.2.1. Killer Klown

- USE `Batteries` with Dexter
- EXIT east (`Broken Ruler`)

#### 14.2.2. Capt'n Jolly's Blow Hole

- EXIT west

### 14.3. Arkade

#### 14.3.1. Arcade Entrance

- PICK UP small hole (`Quarter`)

#### 14.3.2. Arcade Back Room

- USE `Quarter` with Drunken Violent Bums
- INTERACT WITH with Drunken Violent Bums
  - USE the cursor keys and `z` and `x` to fight and `c` and `v` to block and unblock
  - WIN the fight
  - QUIT
- INTERACT WITH **all three** coin return trays (`Dime`)
- USE `Dime` with coin slot (in the Bubble Joy Fun machine)
- PICK UP bubble gum (`Piece of Gum`)
- EAT `Piece of Gum` (`Chewed Bubble Gum`)
- USE `Chewed Bubble Gum` with `Broken Ruler` (`Repaired Ruler`)

### 14.4. Mini-Golf Course

#### 14.4.1. Capt'n Jolly's Blow Hole

- EXIT east

#### 14.4.2. Jungle Boogie

- LOOK AT dark bushes (on the right)
- USE `Mirror` with dark bushes

### 14.5. Crosswalks

#### 14.5.1. Stickz & Ballz

- PICK UP rags (on the ball washer) (`Rag`)
- USE `Rag` with `Mirror`

### 14.6. Mini-Golf Course

#### 14.6.1. Jungle Boogie

- USE `Mirror` with dark bushes (`White Ball`)

### 14.7. Arkade

#### 14.7.1. Arcade Back Room

- USE `White Ball` with Nora

## 15. Punks

### 15.1. Arkade

#### 15.1.1. Arcade Entrance

- PICK UP small hole (`Quarter`)

#### 15.1.2. Arcade Back Room

- USE `Quarter` with Drunken Violent Bums
- INTERACT WITH with Drunken Violent Bums
  - USE the cursor keys and `z` and `x` to fight and `c` and `v` to block and unblock
  - WIN the fight
  - QUIT
- INTERACT WITH **all three** coin return trays (`Dime`)
- USE `Dime` with coin slot (in the Bubble Joy Fun machine)
- PICK UP bubble gum (`Piece of Gum`)
- EAT `Piece of Gum` (`Chewed Bubble Gum`)

### 15.2. Mini-Golf Course

#### 15.2.1. Killer Klown

- USE `Chewed Bubble Gum` with klown hole (below the clown)
- EXIT east

#### 15.2.2. Capt'n Jolly's Blow Hole

- EXIT east

#### 15.2.3. Jungle Boogie

- TALK TO Ivan Trash
- TALK TO Gunther Grunge
- EXIT west

#### 15.2.4. Capt'n Jolly's Blow Hole

- TALK TO Nora

### 15.3. Haunted House

#### 15.3.1. Mirror Room

- WALK left
- USE `Chainsaw` with ghost
- EXIT east

### 15.4. Mini-Golf Course

#### 15.4.1. Jungle Boogie

- USE `Mirror` with dark bushes (`White Ball`)

### 15.5. Arkade

#### 15.5.1. Arcade Back Room

- USE `White Ball` with Gunther Grunge

### 15.6. Crosswalks

- INTERACT WITH door
