# This odd Feeling - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 17.05.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/this-odd-feeling)

## 1. Campsite

- EXIT north

## 2. Village

- TALK TO fortune teller
  - "Yes"
- TALK TO fruits and vegetable guy
  - TALK about weather
    - When he talks about the farmers complaining, CLICK on the farmers card (middle card) (100/100)
- EXIT west

## 3. Farmers

- TALK TO Farmer Jake (10/110)
- EXIT west

## 4. Village

- TALK TO fruits and vegetable guy
  - "Talk about the farmers"
    - When he doesn't want to tell you, USE Blood feud card (100/210)
- TALK TO Hector
- EXIT west

## 5. Farmers

- TALK TO Farmer Jake
  - "Yes, them!"
  - "Yes, briefly"
  - "He wants to apologize!" (10/220)
- EXIT west, northeast

## 6. Campsite

- EXIT north

## 7. Village

- TALK TO Hector

## 8. Farmers

- TALK TO Farmer Jake  (10/230)

## 9. Village

- TALK TO Frank
- USE old flute (on his stand)
- USE `Wallet` with old flute (`Flute`)

## 10. Farmers

- USE `Flute` with Farmer Jake (10/240)
- TALK TO Farmer Jake
- PICK UP bucket (on the right side of the tractor) (`Bucket`)

## 11. Village

- TALK TO Rosa
  - "Tell her about the dirty tractor"

## 12. Campsite

- USE backpack (`Rosa's Sponge`) (10/250)
- USE travel bag (`Dish Soap`, `Knife`, `My Sponge`)

## 13. Village

- USE `Bucket` with pump base (of the red pump)
- USE water pump
- PICK UP bucket (`Bucket with Water`)

## 14. Farmers

- USE `Dish Soap` with `Bucket with Water` (`Bucket with Soapy Water`)
- USE `Rosa's Sponge` with tomato goo (on the front of the tractor) (`Bucket`) (10/260)
- TALK TO Farmer Jake

## 15. Village

- PICK UP barb wire (to the right of the tavern, behind the tree on the left side) (`Barb Wire`)

## 16. Farmers

- USE `Barb Wire` with Farmer Jake (`Tools`)
- PICK UP bag of seeds (`Seeds`)
- USE `Tools` with broken fence (on the right) (10/270)
- TALK TO Farmer Jake

## 17. Village

- TALK TO Hector
- USE `Bucket` with pump base
- USE water pump
- PICK UP bucket (`Bucket with Water`)
- USE `Dish Soap` with `Bucket with Water` (`Bucket with Soapy Water`)
- USE `My Sponge` with smelly residue (on the tavern's window) (`Bucket`) (10/280)
- TALK TO Hector

## 18. Farmers

- TALK TO Farmer Jake (`Rocking Chair`)

## 19. Village

- USE `Rocking Chair` with Hector (10/290)
- TALK TO Hector
- USE `Wallet` with cape (at Frank's Commodities) (`Cape`)
- USE `Seeds` with little yellow bird (in the tree)
- **NOTE:** SAVE YOUR GAME HERE
- WALK behind the big tree
- WALK a few steps to the right (but not too far)
- USE `Cape` with little yellow bird (10/300)
- TALK TO Hector (10/310)

## 20. Farmers

- TALK TO Farmer Jake
  - When he says, it's a long story, USE Monster card (100/410)

## 21. Village

- TALK TO Hector
- USE door (to the tavern)

### 21.1. Tavern

- TALK TO Katia
  - "Talk about the dream."
  - While you ask "what is true?", USE Ancient War card (100/510)
- EXIT south, west (10/520)

## 22. Farmers

- TALK TO Rosa
  - While talking about a potion, USE Chunky Salsa card (100/620)

## 23. Village

### 23.1. Tavern

- TALK TO Katia
- PICK UP bowl (`Large Bowl`)

## 24. Village

- USE `Wallet` with tomatoes
  - "Five tomatoes please" (`Tomatoes`)
- USE `Wallet` with green peppers (`Green Peppers`)
- USE `Wallet` with lemons (`Lemon`)

## 25. Campsite

- USE campfire (`Charcoal`)
- PICK UP snail (on the tree with the right) (`Snail`)
- USE mini cooler (`Beer Can`)

## 26. Village

- USE odd root (at Frank's Commodities)
- USE `Wallet` with odd root (`Ooba Tooba Root`)

## 27. Farmers

- TALK TO Farmer Jake
  - "Ask for an onion" (`Onion`)

## 28. Village

### 28.1. Tavern

- USE `Onion` with `Large Bowl`
- USE `Tomatoes` with `Large Bowl`
- USE `Green Peppers` with `Large Bowl`
- USE `Lemon` with `Large Bowl`
- USE `Snail` with `Large Bowl`
- USE `Charcoal` with `Large Bowl`
- USE `Beer Can` with `Large Bowl`
- USE `Ooba Tooba Root` with `Large Bowl`
- USE `Knife` with `Large Bowl`
- USE `Large Bowl` with Katia (`Chunky Salsa`) (10/630)

## 29. Farmers

- TALK TO Rosa
  - While she talk about her mother and uncle, USE Bearded Man card
