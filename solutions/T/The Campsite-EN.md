# The Campsite - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 01.11.2025 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [itch.io](https://guillemespias.itch.io/the-campsite).

## 1. Garden

- ENTER car
  - USE (left) driver's visor
  - PICK UP envelope (`Envelope`)
  - USE `Envelope` (`Map`, `Mission details`)
  - PICK UP UV flashlight (`UV Flashlight`)
  - >**ACHIEVEMENT:** Mission Details
    >- USE `UV Flashlight` on `Mission details`

## 2. Campsite

- PICK UP loose stake (from the tent) (`Stake`)
- PICK UP log (`Log`)
- PICK UP stone (on the right boulder) (`Stone`)
- EXIT north into the Deep Jungle (`Brushwood`)

## 3. Garden

- USE `Stone` on log (`Logs`)
- EXIT west

## 4. Campsite

- EXIT west

## 5. Creek

- >**ACHIEVEMENT:** Five-star Gazer
  >- LOOK AT **five** shooting stars which appear randomly in the sky
- TAKE log (`Logs`)
- EXIT northeast (or southeast **2 times**)

## 6. Garden

- ENTER car
  - USE shiny thing (on the left side)
  - USE `Stake` on red wire
  - USE alarm switch
  - EXIT
- ENTER car
- PICK UP nest (`Tinder`)
- ENTER car
  - USE lighter plug
  - PICK UP lighter plug (`Heated lighter`)
  - USE `Heated lighter` on `Tinder` (`Glowing tinder`)

## 7. Campsite

- USE `Brushwood` on campfire
- USE `Logs` on campfire
- USE `Glowing tinder` on campfire
- >**ACHIEVEMENT:** Troll Prodigy
  >- USE Garibaldo **8 times**
- PICK UP canteen (at the left chair) (`Canteen`)
- USE backpack (in the tent) (`Car key`)

## 8. Creek

- USE `Canteen` on creek

## 9. Garden

- USE `Canteen` on windshield
- ENTER car
  - USE `Car keys` on keyhole
  - USE wiper switch
  - USE keyhole
  - EXIT
- PICK UP stone (`Stone`)
- ENTER car
  - USE keyhole

## 10. Creek

- EXIT southeast to the campsite

## 11. Garden

- >**ACHIEVEMENT:** Gossip Guru
  >- WAIT and just listen to the party guests until they run out of things to say
- USE sangria glass
- PICK UP talkie-walkie (`Talkie-walkie`)
- USE `Talkie-walkie`

## 12. Campsite

- USE backpack (`Battery`)

## 13. Creek

- >**ACHIEVEMENT:** Environmentalist
  >- USE `Battery` on creek **3 times**
- USE `Battery` on `Talkie-walkie` (`Drained`)

## 14. Garden

- USE `Talkie-walkie` **4 times**
- >**ACHIEVEMENT:** Morse Code Pro
  >- Decode the morse code having listened to it only once
  >- **RIDDLE:** Morse code
  >   - Note the monitor in the background
  >   - The Morse code contains 3 numbers
  >   - ..... (5)
  >   - .---- (1)
  >   - ----- (0)
- USE lock
  - ENTER code you got from the Morse code `510`
- USE cabinet
- PICK UP shovel (`Shovel`)

## 15. Creek

- >**ACHIEVEMENT:** The Treasure
  >- USE `Shovel` on treasure spot
- WALK east
