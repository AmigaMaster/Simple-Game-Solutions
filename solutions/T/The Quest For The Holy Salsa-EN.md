# The Quest For The Holy Salsa - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 02.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/quest-for-the-holy-salsa)

## 1. Town Gate

- EXIT west

## 2. Big Tree

- TALK TO Drunkard (Isaac Newton)
- WALK on front of Isaac
- JUMP
- PICK UP apple (`Apple`)
- PICK UP branch (on the right from the tree) (`Branch`)
- WALK under the left branch
- RAISE tentacle
- JUMP (`Branch`)
- EXIT east

## 3. Town Gate

- WALK to the top of the small hill
- RAISE tentacle
- USE `Apple` on mighty guard
- USE mighty guard
- USE string
- USE `Branch` on string (`Long String`)
- USE mighty guard
- USE `Branch` on (short) string (`String`)
- EXIT east

## 4. Town

- USE `String` on (bend) `Branch` (`Crossbow`)
- USE `Long String` on (straight) `Branch` (`Branch tied to Rope`)
- USE `Branch tied to Rope` on gate
- USE `Crossbow` on string
- RAISE tentacle
- JUMP
- USE temple door

## 5. Temple

### 5.1. Entrance Door

- USE door
- TALK TO idol
- USE plant (`Love Letter`)
- EXIT west **3 times**

## 6. Big Tree

- EXIT west

## 7. Cave

- TALK TO the high one
- USE `Love Letter` on the high one

## 8. Town

- RAISE tentacle
- JUMP
- USE `Love Letter` on second farmer from the right (`Red Chili Pepper`)

## 9. Big Tree

- TALK TO Isaac Newton **2 times**
- USE `Red Chili Pepper` on Isaac Newton
- TALK TO Isaac Newton

## 10. Temple

### 10.1. Entrance Door

- TALK TO idol
  - "All beards will hit the ground..."
- **ATTENTION:** SAVE YOUR GAME HERE
- PICK UP holy chunky salsa
- **QUICK:**
  - JUMP
  - USE table
  - USE paintings
