# Technobabylon Freeware Part 3: In Nuntius Veritas - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 22.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

## 1. Holding Area

- CONNECT TO TRANCE (in your inventory bar) (`Wetware.HLP`)
- USE Comm. Terminal (`Food Machine Token`)
- USE top bunk
- PICK UP cup (`Cup`)
- USE `Cup` on water dispenser (`Cup of Water`)
- USE `Food Machine Token` on food machine (`"Food"`, `Utensil`)
- USE `"Food"` on `Cup of Water`
- USE `Wetware Residue` on `Cup of Water`
- USE table
- USE `"Food"` on camera
  - CLICK when the cursor is in the middle
- USE `Utensil` on guy (`Piece of Plastic`)
- USE `Documents` on wastepaper bin
- USE table
  - USE `Piece of Plastic` on `Utensil` (`Rubber-covered Utensil`)
  - USE `Rubber-covered Utensil` on broken light
  - USE `Documents` on broken light (`Burning Paper`)
  - USE `Burning Paper` on wastepaper bin (`Wetware Module`)
- CONNECT TO TRANCE via network terminal

## 2. Trance

- Repeat the blinking sequence  **3 times**

## 3. Holding Area

- CONNECT TO TRANCE via guy

## 4. Trance

- Fight the red cells until the last one - Simply keep clicking (`Epinephrine`)

## 5. Holding Area

- USE Comm. Terminal
- USE medical drone (`Injector`, `Scalpel`)
- USE `Scalpel` on food machine (`Memory Module`, `Optical Module`, `Power Module`)
- LOOK AT `Power Module`
- USE table
  - USE `Power Module` on broken light
  - USE table
- USE `Memory Module` on `Epinephrine`
- LOOK AT `Optical Module`
- USE `Food Machine Token` on bolts (at the base of the table)
- USE table
- USE (left) table edge **2 times** (to move the table under the camera)
- USE table
  - USE `Scalpel` on camera (`Lens`)
  - USE `Lens` on `Optical Module`
  - USE table
- USE `Power Module` on food machine
- USE `Optical Module` on food machine
- USE `Memory Module` on food machine
- USE `Food Machine Token` on food machine
- USE `Injector` on guy
- CONNECT TO TRANCE via guy (`Program`)
- USE network terminal
  - DRONE/UTL
  - For each level, you need to move the drone to the square, download code and move to the red square. But you need to give all instructions in advance. L=Left, R=Right, U=Up, D=Download
    1. L, U, R, 4xU, R, 2xU, L, U, D, 2xR, U, L, 4xU, R, 4xU, Execute (25 instructions)
    2. R, 2xU, R, 3xU, L, 2xU, R, U, L, 2xU, L, 3xU, R, U, L, D, L, U, L, 3xU, R, 2xU, R, 5xU, R, U, L, U, R, 2xU, Execute (44 instructions)
    3. 2xR, U, L, 6xU, L, 2xU, L, 2xU, R, 3xU, R, 3xU, L, U, D, 2xL, U, R, 3xU, R, U, Execute (36 instructions)
- USE drone
- TALK TO guy (`Large Plastic Sheet`)
- USE `Large Plastic Sheet` on drone
- USE drone (`Cutter`)
- USE (right) table edge **2 times**
- USE `Cutter` on vent
