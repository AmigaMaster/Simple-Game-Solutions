# The Void - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 23.01.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [itch.io](https://digital-mosaic-games.itch.io/the-void)

## 1. Ship

- USE switch
- USE wires
- USE switch
- USE ship computer
  - Instructions
    - **NOTE:**
      1. Boot Up Radar
      2. Boot Up NavCom
      3. Locate Anomaly
      4. Return Home
  - Data
    - **NOTE:** Coordinates `36-08-29`
  - Personnel
    - **NOTE:** `Dean Wilco`
  - USE red button on the monitor to exit
- USE NavCom
  - Radar
  - Dean Wilco
  - 36-08-29

## 2. Anomaly

- PICK UP Analyzer v1.7
- USE Anomaly
  - Analyze
  - Data
  - Hazard Check
  - Should we enter?
  - Enter the Portal

## 3. Inside Anomaly

- USE Gate
  - Analyze carving
  - Writing in the middle
  - Introduction
  - How come I understand you?
  - What is this?
  - Tell me about the Portal
  - What now?
  - Choose as you like - there are two different endings
