## 1. Game

- TALK TO Jess
- PICK UP branch (`Stick`)
- USE corpse (`Handkerchief`)
- USE `Handkerchief` on `Stick` (`Flag`)
- PICK UP rubble (right from the corpse) (`Rubble`)
- USE Flag on raven
- USE car **2 times** (`Lead`)
- PICK UP rag (on the right side next to Jess) (`Oily Rag`)
- PICK UP clamp (on the ground above the panel) (`Clamp`)
- USE `Rubble` on panel
- USE panel
- USE `Oily Rag` on panel
- USE panel (to turn the power on)
- USE pump
- USE panel (to turn the power off)
- USE `Lead` on pump
- USE panel (to turn the power on)
- USE pump
- USE panel (to turn the power off)
- USE `Clamp` on `Oily Rag`
- USE `Clamp` on pipe
- USE panel (to turn the power on)
- USE pump
- TALK TO Jess
- USE painting
