## 1. Evil Mountain

- ENTER ITV building

### 1.1. ITV

#### 1.1.1. Reception

- USE `Credit Card` on elevator

#### 1.1.2. Second Floor

- USE telephone wire
- USE bag (of the engineer) (`Wire Cutters`)
- USE `Credit Card` on elevator

#### 1.1.3. Reception

- USE `Wire Cutters` on loose wire (`Wire`)
- USE `Credit Card` on elevator

#### 1.1.4. Second Floor

- USE `50p on a Rope` on vending machine
- PICK UP picture
- USE `Wire` on hook
- USE wire
- USE `Candy Cane` on guard
- USE `Wire Cutters` on wire
- USE `Credit Card` on (right) elevator

#### 1.1.5. Laboratory

- TALK TO scientist
- EXIT west

#### 1.1.6. Teleporter Room

- USE HyperCupboard TM (`Lightsaber`, `Blackmail Worthy Photo`, `Plank Avec Nail`, `First Strike Capability`, `Santa Voodoo Doll`, `FF7 SUPER SECRET ATTACK`)
- EXIT east

#### 1.1.7. Laboratory

- USE `Lightsaber` on scientist
