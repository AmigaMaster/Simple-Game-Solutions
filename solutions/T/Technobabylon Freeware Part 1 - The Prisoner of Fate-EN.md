# Technobabylon Freeware Part 1: The Prisoner of Fate - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 21.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

## 1. Trance

- USE exit button

## 2. Your Room

- USE (right) door
- USE terminal (beside the door)
- LOOK AT toilet
- USE toilet (`Wetware Node`)
- USE `Wetware Node` on terminal
- USE uplink terminal (at the bedside)

## 3. Trance

- USE mail buffer
- USE datapack (`Virus Pack`)
- TALK TO knight
- USE exit

## 4. Your Room

- PICK UP food tray (`Plastic Fork`, `Plastic Tray`)
- USE `Plastic Fork` on terminal
- USE left locker (under the bed)
- PICK UP shower gel (`Shampoo`)
- USE `Shampoo` on left locker (`Trophy`, `Cutters`)
- USE `Trophy` on food machine
- USE food machine
- USE memory slot (in the food machine) (`Data Card`)
- USE `Wetware Node` on food machine
- USE `Data Card` on uplink terminal

## 5. Trance

- TALK TO Cheffie (on the left)
- USE `Virus Pack` on memory card
- USE exit

## 6. Your Room

- USE `Data Card` on memory slot
- USE uplink terminal

## 7. Trance

- TALK TO Cheffie
  - "What's on your recipe card now?"
- TALK TO knight
  - "I think Cheffie might have an infection."

## 8. Your Room

- USE food machine
- USE `Cutters` on shower hose (`Shower Hose`)
- USE `Shower Hose` on food machine
- USE `Cutters` on cable (at the pillar) (`Cable`)
- USE `Cable` on grille (above the door) (`Grille`)
- USE `Grille` on food machine
- USE food machine (`Metal Fork`, `Metal Tray`)
- USE `Metal Fork` on terminal
