# The Secret Of Chunky Salsa - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------
|   1.0   | 14.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/secret-of-chunky-salsa-v1.3)

## 1. Bearders' Village

### 1.1. Street

- TALK TO Clarke
- TALK TO Joshua
- USE bakery (`Bread`)
- USE `Bread` on magpie
- PICK UP magpie
- PICK UP shiny thing (in the breadcrumbs) **2 times** (`Small Key`)
- TALK TO Joshua (`Tomato`)
- TALK TO Clarke
- ENTER Chuck's house

#### 1.1.1. Chuck's House

- USE `Small Key` on alarm clock
- USE alarm clock (`Great Artifact`)
- EXIT house

### 1.2. Street

- ENTER home

#### 1.2.1. Home

- TALK TO father
- PICK UP ink (`Ink`)
- EXIT home

### 1.3. Street

- TALK TO Clarke **2 mal**
- EXIT north

## 2. Map

- WALK TO Moustachine's Village

## 3. Moustachine's Village

- USE gate
- EXIT north

## 4. Druid's House

- TALK TO druid
- EXIT east

## 5. Festival Field

- USE cauldron **3 times**
- PICK UP weed (`Weed`)
- EXIT southwest

## 6. Bearders' Village

### 6.1. Street

#### 6.1.1. Home

- USE `Weed` on drying table (`Dry Weed`)

## 7. Druid's House

- USE `Dry Weed` on druid
- USE `Ink` on druid (`Invisible Ink`)
- USE `Invisible Ink` in Colin (`Empty Bottle`)

## 8. Moustachine's Village

- USE gate
- USE bell
- USE `Great Artifact` on bell

### 8.1. Bakery

- "...lost your daughter..."
- "Mr. Jasper (lanky)"
- USE table
  - **NOTE:** One of the cooks is lying and giving wrong descriptions, so use the following ingredients
  - USE pepper (top left)
  - USE leek (top, right to the mushrooms)
  - USE garlic (right)
  - USE onion (right)
  - USE apple vinegar (lower right)
  - USE salt and pepper (lower right)
  - USE tomato (bottom) (`Dish`)
- USE `Dish` on cooks (`Chunky Salsa`)
- EXIT south

## 9. Festival Field

- ENTER beam of light

## 10. Druid's House

- TALK TO Druid
- USE `Empty Bottle` on lake (`Water`)
- USE `Water` on Colin

## 11. Festival Field

- ENTER beam of light

### 11.1. Spaceship

- LOOK AT door
- PICK UP pipe (right to the shower door) (`Pipe`)
- USE `Pipe` on door
- USE (red) button (`Girl`)
- **ATTENTION:** SAVE YOUR GAME HERE
- PICK UP element (in the device in the middle of the room)
- PICK UP L.O.V.E. (`L.O.V.E.`)
- ENTER room

#### 11.1.1. Cockpit

- USE `L.O.V.E.` on Power node (the pink tube on the left)
- USE the three handles and set up the following
  - Top handle: Middle
  - Middle handle: left
  - Bottom handle: right
- USE lever (on the left)
- USE handle (the steering with the red button)
