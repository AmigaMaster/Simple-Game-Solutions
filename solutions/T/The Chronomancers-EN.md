# The Chronomancers - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 10.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [archive.org](https://archive.org/details/the-chronomancers)

## 1. Cable Street

### 1.1. Home

- WALK left
- USE message tube
- WALK right to leave the house
- EXIT east

### 1.2. Train Station

- TALK TO irritating official (behind the desk)
  - "I would like to purchase a ticket." (`Ticket`)
- WALK right
- USE train

## 2. Phoenix Street

- USE speaking tube
- USE elevator
- PICK UP screwdriver (from the table) (`Screwdriver`)
- USE elevator
- EXIT east

### 2.1. Train Station

- WALK right
- USE train

## 3. Dorian Road

- USE speaking tube
- PICK UP rope (`Rope`)
- USE `Screwdriver` with brick (in the checkered road) (`Brick`)
- USE `Rope` with basket
- USE `Brick` with basket and rope
- USE basket, rope and brick **2 times**
- USE contraption (`Mysterious Page`)
- LOOK AT `Mysterious Page`
- EXIT west

## 4. Phoenix Street

- USE elevator
- USE `Mysterious Page` with Vincent Chronotock
- USE elevator
- EXIT east

## 5. Cord Street

- PICK UP umbrella (`Umbrella`)
- PICK UP blunderbuss (`Blunderbuss`)
- PICK UP ray gun (`Ray Gun`)
- USE Escape Route (door)

## 6. Cable Street

- USE `Umbrella` with irritating official
- PICK UP inkstand (`Inkstand`)

## 7. Phoenix Street

- USE elevator
- USE `Inkstand` on Vincent Chronotock (`Page (with notes)`)
- LOOK AT `Page (with notes)`
- USE elevator

## 8. Cable Street

- EXIT west

### 8.1. Home

- WALK left
- TALK TO Amelia Raccoon
- USE `Blunderbuss` with Amelia Raccoon

### 8.2. Library

- USE door

#### 8.2.1. Inside Library

- USE trashy romance novels (`Pile of Trashy Books`)
- USE completely obvious trapdoor (beside the shelf on the left)

##### 8.2.1.1. Secret Room

- PICK UP petrol can (`Empty Petrol Can`)
- USE door

#### 8.2.2. Inside Library

- USE door

### 8.3. Library

- USE `Empty Petrol Can` with motor car (`Can Full of Petrol`)

#### 8.3.1. Inside Library

##### 8.3.1.1. Secret Room

- USE `Can Full of Petrol` with slot (on the left)
- USE `Pile of Trashy Books` with fuel container (next to the slot)
- USE switch (next to the fuel container)
- USE harpsichord
- Try some of the dialog options. The correct sequence:
  - "Do you know what this machine is?"
  - "Computer, what do you think of all this?"
  - "I think it's a rather..."
  - "It's Milquetoast"
