## 1. Tower South

- USE shears (under the rock) **2 times** (`Shear Blade`)
- USE toolbox (`Adhesive Tape`, `Telescopic Rake`)
- USE `Adhesive Tape` on `Shear Blade` (`Shear covered in tape`)
- USE `Telescopic Rake` on `Shear covered in tape` (`Telescoped Shear`)
- USE `Telescoped Shear` on young woman
- USE young woman **2 times** (`Cable`)
- USE `Cable` on young woman
- USE petals (on the left edge) (`Vodka`)
- USE `Vodka` on young woman
- USE overshirt (on the rock) (`Shirt`)
- USE `Multitool` on `Shirt` (`Ribbon`)
- USE `Ribbon` on young woman

## 3. Summerglow Apartments

### 3.1. Living Room

- TALK TO maid
- EXIT upstairs

### 3.2. Bedroom

- LOOK AT (right-click) body
- LOOK AT pool
- USE pool
- LOOK AT wine glass (on the floor right next to the pool)
- USE wine glass
- LOOK AT hand (in front of the bed on the floor)
- USE hand (`Hand`)
- USE hanger (on the ground in front of the bed) (`Coathanger`)
- USE photoframe (right of the bed) (`Photograph`)
- LOKO AT `Photograph`
  - **NOTE:** `Paris, June, '86`
- USE controls (next to the door)
- USE `Coathanger` on controls
- USE controls
  - Outdoor
  - USE fishing rod (`Fishing Rod`)
  - DONE
- USE `Fishing Rod` on pool (`Gauss Pistol`)
- EXIT east

### 3.3. Living Room

- TALK TO food machine
- TALK TO drink machine
- USE kitchen knife (`Knife`)
- USE `Knife` on maid (`Power Cell`)

### 3.4. Bedroom

- USE `power Cell` on charger (on the left of the bed)

### 3.5. Living Room

- USE `Power Cell` on maid
- TALK TO maid
- LOKO AT maid
- USE terminal
  - CLICK Forgotten? (Hint: `Our special place`)
  - ENTER `paris`
- USE heating gel (on the counter) (`Cooking Gel`)
- LOOK AT `Cooking Gel`
  - Temperature `37C`
  - CONFIRM
- USE `Cooking Gel` on `Hand`
- USE `Hand` on terminal
- USE terminal
  - **NOTE:** You need to have looked at the maid in order to call the vendor
- TALK TO maid (it does not answer)
- USE terminal (`CI Splitter`)
- USE `CI Splitter` on maid (`Memory Unit: Maid`, `Personality: Maid`, `Role Memory: Maid`)
- USE `CI Splitter` on food machine (`Memory Unit: Cheffie`, `Personality: Cheffie`, `Role Memory: Cheffie`)
- USE `Personality: Cheffie` on maid
- USE `Memory Unit: Maid` on maid
- USE `Role Memory: Maid` on maid
- TALK TO maid
- USE `Gauss Pistol` on maid
- USE `CI Splitter` on maid (`Memory Unit: Maid`, `Personality: Cheffie`, `Role Memory: Maid`)
- USE `CI Splitter` on drink machine (`Memory Unit: Barkeep`, `Personality: Barkeep`, `Role Memory: Barkeep`)
- USE `Memory Unit: Maid` on maid
- USE `Personality: Cheffie` on maid
- USE `Role Memory: Barkeep` on maid
- USE `Gauss Pistol` on maid (`Bone`)
