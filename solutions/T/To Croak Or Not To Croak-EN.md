# To Croak Or Not To Croak - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 08.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [itch.io](https://esmeralda.itch.io/to-croak-or-not-to-croak)

## 1. Cottage

### 1.1. Inside

PICK UP crystal ball (`Broken Crystal Ball`)
- TALK TO raven
  - "Come down here..." (`Raven`)
- EXIT cottage

## 2. Cottage

- PICK UP tree resin (from the tree on the left) (`Tree resin`)
- PICK UP hellebore (`Hellebore`)
- USE `Raven` on blackbird's nest (`Blackbird Egg`)
- USE `Yellow Potion` on `Raven` (`Frog`)
- USE `Frog` on henbane (`White Henbane`)
- USE `Frog` on billy goat
- PICK UP rock salt (`Rock Salt`)
- PICK UP pile of wood (at the house's edge) (`Wood`)
- EXIT north

## 3. Mill

- PICK UP devil's trumpet (`Devils Trumpet`)
- PICK UP yellowcress (`Watercress`)
- LOOK (right click)  AT window
- USE window
- EXIT east

## 4. Cottage

- PICK UP frog

### 4.1. Inside

- USE chest
  - Set flag, moon and the symbol which looks like `><` 90 degrees rotated (`Grimoire`)
- USE `Grimoire`
  - **NOTE:** The ingredients for the transformation potion and the dependencies as well as the replacements
- USE `Wood` on oven
- LOOK AT (right-click) firewood
- USE kettle
  1. USE blackbird egg on cauldron
  2. USE belladonna on cauldron
  3. USE white henbane on cauldron (`Red Potion`)
- USE `Red Potion` on `Frog` (`Mouse`)
- USE `Mouse` on hole (`Piece of Crystal Ball`)
- USE `Piece of Crystal Ball` on `Tree Resin` (`Piece with tree resin`)
- USE `Piece with tree resin` on `Broken Crystal Ball`
- USE kettle
  1. USE toad slime on cauldron
  2. USE devil's trumpet on cauldron
  3. USE rock salt on cauldron (`Blue Potion`)
- USE `Blue Potion` on `Mouse` (`Elfant`)
- USE crystal ball
  - USE crystal ball

## 5. Mill

- USE `Elfant` on door
- USE window (`Comb with Hair`, `Raven`)

## 6. Cottage

### 6.1. Inside

- USE kettle
  1. USE comb with hair on cauldron
  2. USE watercress on cauldron
  3. USE rock salt on cauldron (`Potion`)
- EXIT cottage and walk to the mill
