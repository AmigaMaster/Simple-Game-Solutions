# Tropic of Capricorn - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------------------------------------------
|   1.0   | 26.03.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/tropic-of-capricorn)

## 1. Main Deck

- TALK TO Ronald
- EXIT upstairs

## 2. Saloon

- TALK TO Captain Swift
- EXIT south

## 3. Main Deck

- TALK TO Ronald
  - Offer to help him
  - ENTER "beer"
- Repeat these steps with the following answers:
  - "cherry"
  - "show"
  - "pole"
- TALK TO Ronald **again**
- USE ladder (in the lower right)

## 4. Lower Deck

- TALK TO Mr. Sharp
  - "I have a message.."
  - "Yes, indeed."
  - "Yes, very amiable."
  - "No, he didn't say that."
  - "Yes, he called you that a lot!"
  - "Everything but pretentious!"
  - "Highly amorous!"
  - "I can't recall him saying that."
- EXIT down

## 5. Main Deck

- TALK TO Ronald

## 6. Saloon

- TALK TO Captain Swift
- TALK TO Sammy
  - Offer help
  - "blue"
  - "dream"
  - "sea"
  - "bright" (`Poem`)
- PICK UP jar of nuts (from the table beside the Captain) (`Nuts`)

## 7. Main Deck

- USE (top left) ladder

## 8. Post

- USE `Poem` on Gloria
- USE telescope
  - EXIT
- USE ladder

## 9. Main Deck

- USE rope-ladder

### 9.1. Mast

- PICK UP message (`Message`)

## 10. Main Deck

- USE door

## 11. Captain's Cabin

- USE `Nuts` on (small) table

## 12. Post

- PICK UP white paint (`White Paint`)
- USE `White Paint` on pipe

## 13. Captain's Cabin

- PICK UP Al the parrot (`Parrot`)
- USE `Message` on `Parrot`

## 14. Saloon

- USE `Parrot` on Captain Swift
