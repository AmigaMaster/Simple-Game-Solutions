# Crackwell Unhinged - Walkthrough

| Version | Date       | Author
|:-------:|------------|-------
|   1.0   | 15.02.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [archive.org](https://archive.org/details/crackwell2)

## 1. Home

- USE puke (`Crack Rock`)
- USE `Crack Rock` on inheritance
- EXIT east

## 2. Neighbor's

- TALK TO Hecanteat Schwarma
- EXIT west

## 3. Home

- TALK TO Blowey

## 4. Neighbor's

- TALK TO Hecanteat Schwarma
- PICK UP pet rock (`Pet Rock`)
- USE `Pet Rock` on grinder (`Ground Rock`)
- EXIT west

## 5. Work

- TALK TO The Madam (`Poison Cocaine`)
- USE `Ground Rock` on `Poison Cocaine`
