# Living Nightmare: Endless Dream - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 30.12.2025 | Amiga Master (AmigaMaster-NG@t-online.de) |

Download the free game from [Archive.org](https://archive.org/details/living-nightmare-endless-dreams)

## 1. Office Building

- USE `Keys` on glass doors
- USE intercom
- USE `Wallet` on window

### 1.1. Hallway

- USE break room (door)

#### 1.1.1. Break Room

- USE `Wallet` on coffee vendor (`Coffee`)

### 1.2. Hallway

- USE `Coffee` on security guard
- USE call button (elevator)

### 1.3. Office

- USE light switch
- USE window (`Supplies Key`)
- USE `Supplies Key` on locked supply cabinet

## 2. Psychiatrist Office

- Choose any answer

## 3. Library

### 3.1. Hallway

- TALK TO nerdy guy
- USE student lounge (door)

#### 3.1.1. Student Lounge

- LOOK AT notice
- USE cabinets (`Scissors`)
- USE phone
- USE door

### 3.2. Hallway

- USE staff room (door)

#### 3.2.1. Staff Room

- PICK UP keys (`Security Keys`)
- LOOK AT notice
- USE door

#### 3.2.2. Student Lounge

- USE `Security Keys` on fire alarm

### 3.3. Hallway

- USE computer
- USE `Scissors` on blue curtains (`Curtain`)
- USE library entrance (doors)

## 4. Grandparents House

### 4.1. Entrance Hall

- EXIT west

### 4.2. Second Floor

- USE grandparents room

#### 4.2.1. Grandparents Room

- PICK UP locket (from bed) (`Silver Key`)
- USE door

### 4.3. Entrance Hall

- USE basement door

#### 4.3.1. Basement

- USE gateway

### 4.4. Second Floor

#### 4.4.1. Grandparents Room

- USE door

### 4.5. Second Floor

- USE `Silver Key` on ??? (left door)

#### 4.5.1. Unused Room

- PICK UP exposed springs (`Spring`)
- LOOK AT cracks
- USE `Keys` on cracks (`Wooden Box`)
- USE `Spring` on `Wooden Box` (`Grandpa's Journal`)
- LOOK AT `Grandpa's Journal`
- USE door

## 5. Graveyard

- USE gate

### 5.1. Mausoleum

- TALK TO door

### 5.2. Dark Room

- USE torch (on the left side)
- USE caskets (in the center of the screen)
