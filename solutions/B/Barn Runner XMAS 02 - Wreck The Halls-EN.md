# Barn Runner XMAS 02 - Wreck The Halls - Walkthrough

| Version | Date       | Author                                    |
|:-------:|------------|-------------------------------------------|
|   1.0   | 02.06.2024 | Amiga Master (AmigaMaster-NG@t-online.de) |

Get the free game from the [Homepage](https://www.barnrunner.com/xmas2.html)

## 1. Parking Lot

- >EASTER EGGS:
  >- USE mailbox (`Letters`)
  >- EXIT northeast to the well
  >- TALK TO robot (`Two disks`)
  >- ENTER outhouse
  >- TAKE magazine (`Girly mag`)
  >- LOOK AT 18 wheeler **multiple times**
  >- LOOK AT monster truck **multiple times**
  >- LOOK AT mailbox **multiple times**
- ENTER church

### 1.1. Church

- Walk to pastor

#### 1.1.1. Pulpit

- >EASTER EGG:
  >- USE `Girly mag` with pastor
  >- TALK TO pastor
  >   - Ask for the magazine
- TALK TO pastor

## 2. Parking Lot

- EXIT north east

### 2.1. Outhouse and Well

- ENTER outhouse
  - TAKE matches (`Matches`)
  - TAKE book (`Appointment book`)
  - TAKE notes (`Sermon notes`)
- >EASTER EGGS:
  >- LOOK AT `Appointment book` **multiple times**
  >- USE `Matches` with frozen water in toilet
  >- USE toilet
- EXIT southeast

## 3. Parking Lot

- TAKE package (`Paper Package`)
- USE mailbox (`Manilla envelope`, `Package`)
- USE `Multi tool` with `Paper package` (`Note`, `Underwear`)
- USE `Multi tool` with `Manilla envelope`(`DuraChip`, `Invoice`)
- USE `Multi tool` with `Package` (`Mulberro Red`, `Mulberro Green`, `Mulberro Silver`, `Mulberro Gold`, `Mulberro candy sticks`, `Seasons Greetings letter`)

### 3.1. Church

- Walk to pastor

#### 3.1.1. Pulpit

- USE `Appointment book` with pastor
- TAKE yule log (`Yule log`)
- TAKE bread (`Bread`)
- EXIT upstairs

#### 3.1.2. Upstairs

- >EASTER EGGS:
  >- LOOK AT each painting **multiple times**
- OPEN (right) door
- USE `Mulberro candy sticks` with skunk
- ENTER (right) door

##### 3.1.2.1. Bedroom

- >EASTER EGGS:
  >- TAKE skunk (`Sleeping skunk`)
  >- USE bed (`Girly mag`, `Phogbound mag`)
  >- USE `Bread` with toaster **7 times** (`Toast`, `Toast`, `Toast`, `Toast`, `Toast`, `Toast`, `Toast with dark spot`)
  >- **NOTE:** This special toast gives you an extra life at an action sequence
  >- LOOK AT stained glass **multiple times**
  >- LOOK AT bookshelf **multiple times**
  >- LOOK AT each of the three carpets **multiple times**
- USE `Yule log` with fireplace
- USE `Matches` with log
- USE `Underwear` with fireplace (`Warm underwear`)
- USE `Mulberro Red` with fireplace (`Tar`)
- USE bookcase
  - >EASTER EGGS:
  >- "A heavy brown book" (`NRA bible`)
  >- USE `Multi tool` with `NRA bible` (`.45 Pistol`)
  >- **NOTE:** This will make an action sequence easier
  >- "A small, worn black book" (`Song of Solomon`)
  >- "A small, old yellow book" (`Tijuana Bible`)
  - "A really big green book." (`Binary bible`)
  - "A red book." (`KJV Bible`)
- USE `Binary bible` with `KJB bible` (`Translated binary phrases`)
- **INFO:** You can throw away all other books, papers and packs of cigarettes by use them on the fireplace.

## 4. Parking Lot

- TAKE newspaper (`Newspaper`)
- >EASTER EGG:
  >- LOOK AT `Newspaper` **multiple times**

### 4.1. Outhouse and Well

- USE crank
- USE `Tar` with crank
- USE rope

#### 4.1.1. Well Shaft

- EXIT west

#### 4.1.2. Dark Room

- >EASTER EGG:
  >- USE `Toast` with blue computer to get rid of the toast in your inventory
- USE `DuraChip` with blue computer
- USE `Warm underwear` with `DataLink`
- TAKE box with yellow robot head (`MindLog disks`)
- USE `DataLink` with blue computer

### 4.2. Church

#### 4.2.1. Upstairs

- USE `Newspaper` with (left) door
- USE `Multi tool` with (left) door
- TAKE key (`Key`)
- USE `Key` with (left) door
- ENTER (left) door (`Lit match`)

#### 4.2.2. Monk's Room

- >EASTER EGG:
  >- TAKE robe (`Ski mask`)
- USE `Lit match` with welder in the bottom right corner
- **ATTENTION:** SAVE YOUR GAME HERE!
- USE radio on the table
  - USE radio
  - Choose any answer

## 5. Action Sequence

- Just keep shooting at the robot and try not to miss
- If yu have the holy toast, this sequence is much easier to win
- If you have the .45 pistol, you can miss some shots
