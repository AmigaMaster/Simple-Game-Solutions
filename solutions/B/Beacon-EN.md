# Beacon - Walkthrough

|   1.0   | 18.04.2026 | Amiga Master (AmigaMaster-NG@t-online.de)

Download the free game from [Adventure Game Studio](https://www.adventuregamestudio.co.uk/play/game/1403-beacon)

## 1. Main Room

- USE Valentin (`Empty Cup`)
- USE `Empty Cup` on puddle (in front of Valentin) (`Cold Water`)
- USE `Cold Water` on Valentin
- TALK TO Valentin
- USE locker (to the lower right)
  - It needs a code
- TALK TO Valentin
  - "What is the storage locker code?"
    - **NOTE:** `414`
- USE locker
  - ENTER `414` (`Screwdriver`, `Roll of Silicone Tape`)
- PICK UP helmet (on the right side of Valentin) (`Helmet`)
- PICK UP signal buoy (Under the barrel on the left) (`Signal Buoy`)
- LOOK AT `Signal Buoy`
  - EXIT detailed view
- PICK UP cone (next to the signal buoy) (`Cone`)
- USE hatch controls (on the right)
- USE hatch

## 2. Maintenance Shaft

- USE `Screwdriver` on warning lamp (`Light Bulb`)
- LOOK AT `Signal Buoy`
  - USE `Light Bulb` on broken bulb
  - USE `Cone` on intact bulb
- USE `Roll of Silicone Tape` on `Helmet` (`Taped Helmet`)
- USE `Taped Helmet` on repair drone
- USE switch (under the panel)
