## X-mas Tree

- >EASTER EGG:
  >- Change the date of your system to December
  >- USE `present` in the menu bar to open it
- >EASTER EGG:
  >- USE couch
- Place the From top to bottom and left to right:

```
            RED
        BLUE    GREEN
    GREEN   GREY    BLUE
BLUE    RED     GREEN   RED
```

## Soup

- PICK UP all ingredients
- >EASTER EGG:
  >- USE sausages with pot
- USE `olive oil` with pot (yellow bottle)
- USE `bacon essence` with pot (green bottle)
- USE `garlic` with pot (white plate)
- USE `kielbasa sausage` with pot (plate with brown slices)
- USE `vegetable` broth with pot (glass can)
- USE `potatoes` with pot (green bowl)
- USE `parsley` with pot (green glass with green cap)
- USE `thyme` with pot (green glass with brown cap)
- USE `pepper` with pot (white glass)
- USE `salt` with pot (grey glass)
- USE `green beans` with pot (grey bowl)
- USE `condensed cream of mushroom soup` with pot (green can)
- USE `cream of mushroom soup` with pot (red can)
- USE `mushrooms` with pot
